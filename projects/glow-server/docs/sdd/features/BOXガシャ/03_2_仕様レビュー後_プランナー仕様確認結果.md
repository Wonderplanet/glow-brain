# 仕様レビュー後 プランナー仕様確認結果（今回は想定を予想して記載しただけで実際には確認していないので注意）

### BOXラインナップ関連
- **Q1**: レアリティが高いリソース。
- **Q2**: BOXには複数段階あり、ラインナップが変わる。
- **Q3**: 100個

### BOXリセット関連
- **Q4**: ユーザー手動。
- **Q5**: 途中でも手動リセットできる。

### 複数回引きの境界条件
- **Q6**: エラーで引けない（残り個数以上は引けない）

### アイテムA（コスト）関連
- **Q7,8**: イベントプレイで獲得できる専用アイテム。
- **Q9**: アイテムAを150個消費して1回引けるという意味であってる。

### 無限ボックス関連
- **Q10**: 無限ボックスのラインナップは、通常BOXと同じにも違うものにもできるようにする。
- **Q11**: 通常BOXを引き終わった後から。
- **Q12**: 無限ボックスのリセットも手動

### 重複キャラクター関連
- **Q13,14**: 既存のガシャなど全ての機能と同じで、キャラのかけらアイテム。ここで特別気にする必要なし。

### 天井・回数制限関連
- **Q15**: 既存の天井の仕組みとは別として実装する。
- **Q16**: BOXガシャに1日あたりの最大プレイ回数制限はない。

### コスト効率関連
- **Q17**: 既存のマスタ設定で割引考慮できるのでそれと同じマスタテーブル設定で使う。

### イベント期間関連
- **Q18**: 2週間とか1ヶ月とか。その時のイベントによる。
