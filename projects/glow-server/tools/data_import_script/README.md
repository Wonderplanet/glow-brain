## クライアントのデータシートからデータ投入用のクエリを作成するPythonスクリプト
- ディレクトリ
  - client_data
    - クライアントデータの.xlsxファイルを配置する
    - クライアントデータの例：[UserLevelDataList](https://docs.google.com/spreadsheets/d/1HRneFx1IGARkTZjtGwbx9TRCaPsRxtF7jm_vLIDJ-NU/edit#gid=0)
  - db_schema
    - DBスキーマ情報を取得するためのcsvファイルを配置する
    - 各DBテーブルごとに1ファイル
  - outputs
    - スクリプトでできたファイルを出力するところ
    - 出力されるもの
      - 各テーブルごとのデータのcsvファイル (データ確認用)
      - query.txt（スクリプト実行した全テーブルのREPLACE INTOクエリ）
- 環境設定
  - [jupyter/scipy-notebook](https://hub.docker.com/r/jupyter/scipy-notebook)
  - 上記イメージのコンテナを起動すれば、run.ipynbを使えます
  - 参考：https://qiita.com/fuku_tech/items/6752b00770552bf4f46b
- 手順
  - DBのデータをcsvファイルとしてdumpして、db_schemaディレクトリに全て配置する
  - データ投入したいデータがあるシートを.xlsxファイルとしてダウンロードして、client_dataディレクトリに配置する
  - run.ipynbのmainブロックのコードを実行する 
  - outputsの中身は同名なら上書きされます
  - query.txtは毎回上書きされます
