# SDD v2 設計フロー全実行

このコマンドは、SDD v2設計の全段階をオーケストレートします（Stage 2で人間確認が必要）。

## 使用方法

```bash
/api:sdd-v2:00-run-full {機能名}
```

例: `/api:sdd-v2:00-run-full 交換所`

## 実行内容

引数: $ARGUMENTS

**sdd-orchestrator-v2 スキル**を使用して、全段階を自動オーケストレートします。

スキルは各フェーズのサブエージェントを順次起動し、コンテキストウィンドウを分離しながら処理を進めます。

## 機能名について

第1引数として機能名を指定してください。

例：
- `交換所`
- `スタミナブースト`
- `新規ガチャ機能`
