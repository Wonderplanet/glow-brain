# 降臨バトル「誰の依頼だ？」マスタデータ

## 概要

このディレクトリには、降臨バトル「誰の依頼だ？」のマスタデータCSVファイルが含まれています。

## ファイル一覧

### MstAdventBattle.csv
- **説明**: 降臨バトルの基本情報
- **レコード数**: 1件
- **主要データ**:
  - バトルID: `quest_raid_you1_00001`
  - イベントID: `event_you_00001`
  - 開催期間: 2026年2月9日 15:00 ～ 2026年2月15日 14:59
  - 挑戦可能回数: 50回

### MstAdventBattleI18n.csv
- **説明**: 降臨バトルの多言語対応データ（日本語）
- **レコード数**: 1件
- **主要データ**:
  - ステージ名: 誰の依頼だ？
  - 説明文: ボスを倒して高スコア獲得！！

### MstAutoPlayerSequence.csv
- **説明**: 敵出現シーケンスデータ
- **レコード数**: 50件
- **構成**:
  - 要素1-4: 初期配置の敵
  - 要素5-43: Wave 1-6の敵出現パターン（ボス戦含む）
  - グループ変更要素: Wave間の遷移、Wave 6終了後にWave 1へループ

## バトル構成

- **初期配置**: 一般敵の配置
- **Wave 1**: リタ、ダグ（ボス）
- **Wave 2**: ルーク（ボス）と追加敵
- **Wave 3**: ハナ（ボス）
- **Wave 4**: 暗殺者ボス x2
- **Wave 5**: リタ（ボス）、ダグ、ハナ
- **Wave 6**: 複数のボス敵、その後Wave 1へループ

## 技術仕様

作成されたCSVファイルは、`projects/glow-masterdata/`にある本番マスタデータと同じ形式（単一ヘッダー行形式）に従っています。これは`sheet_schema/`のテンプレート形式（3行ヘッダー）とは異なりますが、実際に使用されているマスタデータファイルと一致しています。

シーケンスデータは、既存の`raid_you1_00001`エントリから抽出されており、デプロイ済みのバトル設定との整合性が保たれています。
