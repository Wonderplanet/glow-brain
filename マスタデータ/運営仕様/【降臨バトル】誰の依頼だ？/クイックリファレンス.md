## よくあるミス

### ❌ やりがちなミス

1. **IDの不一致**
   - MstAdventBattle の mst_in_game_id と MstInGame の id が一致していない

2. **開催期間の設定ミス**
   - start_at が end_at より後になっている
   - 日時フォーマットが間違っている（正: `"2026-02-09 15:00:00"`）

3. **多言語の設定漏れ**
   - 日本語だけ設定して英語・中国語を忘れる

4. **報酬アイテムIDの誤り**
   - 存在しないアイテムIDを設定してしまう

### ✅ 必ずチェック

- [ ] 全テーブルのENABLE列が `e` になっている
- [ ] release_keyが統一されている（例: `202602010`）
- [ ] 日時フォーマットが正しい（ダブルクォートで囲む）
- [ ] 多言語テキストが4言語すべて設定されている
