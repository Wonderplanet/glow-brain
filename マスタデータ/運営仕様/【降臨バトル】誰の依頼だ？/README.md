## 📊 必要なマスタデータテーブル

設定が必要なテーブルの一覧：

| テーブル名 | 設定難易度 | 必須度 | 説明 |
|-----------|----------|-------|------|
| MstAdventBattle | ⭐ | 必須 | 降臨バトル本体 |
| MstInGame | ⭐⭐ | 必須 | ゲーム内設定 |
| MstEnemyStageParameter | ⭐⭐⭐ | 必須 | 敵キャラステータス（数が多い） |
