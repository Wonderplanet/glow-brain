# 敵キャラクター設定データ抽出（Sequence.htmlより）

このドキュメントは、Sequence.htmlから抽出した敵キャラクターのデータを、MstEnemyStageParameterの設定用にまとめたものです。

---

## Wave構成

```
Wave 0（初期）: 不良系金髪イケメン × 2 + イケメンじゃない殺し屋 × 2
    ↓
Wave 1: 不良系金髪イケメン × 2 + リタ + ダグ（強敵）
    ↓
Wave 2: 不良系金髪イケメン × 2 + イケメンじゃない殺し屋 × 2 + ルーク（強敵）
    ↓
Wave 3: 不良系金髪イケメン × 4 + ハナ（強敵）
    ↓
Wave 4: 不良系金髪イケメン × 2 + イケメンじゃない殺し屋 × 5（うち1体は強敵）
    ↓
Wave 5: 不良系金髪イケメン × 5 + リタ（強敵）+ ダグ × 2 + ハナ × 2
    ↓
Wave 6: 不良系金髪イケメン × 2（強敵）+ イケメンじゃない殺し屋 × 3（強敵）
    ↓
Wave 1に戻る（ループ）
```

## ゲームバランス調整のポイント

### 難易度の段階的上昇

- **Wave 0→1**: 体力2倍、強敵登場
- **Wave 1→2**: 体力2.5倍、攻撃力大幅UP
- **Wave 2→3**: 体力3倍、強敵ボス登場
- **Wave 3→4**: 体力5倍、ノックバック攻撃
- **Wave 4→5**: 体力7倍、複数強敵同時出現
- **Wave 5→6**: 体力8倍、全敵が強敵化

### 特殊能力

- **リタ**: 攻撃UP（自己バフ）
- **ダグ**: 攻撃DOWN（デバフ）
- **ルーク**: 毒攻撃
- **ハナ**: 高火力アタッカー
- **強敵版**: ノックバック攻撃

### 属性の意図

- **Green（緑）が多数**: リタ（緑属性キャラ）を活躍させるため
- **Colorless**: 最終waveや強敵に使用、属性相性なし

## MstEnemyStageParameter 設定時の注意点

1. **Enemy IDの命名規則**
   - 通常敵: `e_you_00001_you1_advent_Normal_Green`
   - キャラクター敵: `c_you_00001_you1_advent_Normal_Colorless`
   - 強敵: `_Boss_` を含む

2. **HP・攻撃力の係数**
   - Sequence.htmlの「体力係数」「攻撃係数」列を参照
   - MstInGameの係数と掛け合わせて最終値を計算

3. **アビリティID**
   - 特殊能力を持つ敵は mst_unit_ability_id1 を設定
   - 例: ability_you_00001_01（攻撃UPアビリティ）

4. **撃破ポイント**
   - drop_battle_point を設定してスコア計算に反映
   - 強敵ほど高ポイント
