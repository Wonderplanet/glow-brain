# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

**glow-brain** は、GLOWゲームプロジェクト全体のAI支援開発のための**統合コンテキストリポジトリ**。4つの独立したリポジトリを1箇所に集約し、横断的な理解・調査・マスタデータ作成を可能にする。

### projects/ - 参照専用コードベース

`projects/` 配下の4リポジトリは **参照専用**（`.git` 削除済み、実際の変更は各リポジトリで行う）:

| ディレクトリ | 内容 | 技術 |
|------------|------|------|
| `projects/glow-server/` | バックエンドAPI (api/)、管理ツール (admin/) | PHP 8.3, Laravel 11, TypeScript |
| `projects/glow-client/` | ゲームクライアント（スクリプトのみ軽量版） | Unity, C# |
| `projects/glow-masterdata/` | マスタデータCSV | CSV |
| `projects/glow-schema/` | API通信・マスタデータ配信スキーマ定義 | JSON, Ruby |

## セットアップ

```bash
# current_version のプロジェクトをセットアップ（前提: jq インストール済み）
./scripts/setup.sh

# 特定バージョンをセットアップ
./scripts/setup.sh v1.6.1-devld

# バージョン切り替え（ブランチ切り替えのみ）
git checkout v1.6.1-devld
```

バージョンとブランチの対応は `config/versions.json` で管理。

## 主要なデータソース

調査・作業時に参照するパス:

| 目的 | パス |
|------|------|
| DBスキーマ（カラム型・制約・enum値） | `projects/glow-server/api/database/schema/exports/master_tables_schema.json` |
| マスタデータCSV（実データ） | `projects/glow-masterdata/*.csv` |
| クライアントマスタデータクラス（自動生成） | `projects/glow-client/Assets/GLOW/Scripts/Runtime/Core/Data/Data/AutoGenerated/` |
| スキーマ定義 | `projects/glow-schema/Schema/` |
| テーブル命名・ID採番ルール | `domain/knowledge/masterdata/` |

## テーブル・CSVの命名規則

| 種類 | 規則 | 例 |
|------|------|----|
| DBスキーマ | snake_case + 複数形 | `mst_events`, `opr_gachas` |
| CSVファイル | PascalCase + 単数形 | `MstEvent.csv`, `OprGacha.csv` |

**テーブルプレフィックス**:
- `mst_` / `Mst*` — 固定マスタデータ
- `opr_` / `Opr*` — 運営施策・期間限定データ
- `usr_*` — ユーザーデータ
- `log_*` — ログデータ

## domain/ - AI活用基盤

```
domain/
├── raw-data/      # 未加工の一次データ（リリース済みマスタデータ等）
├── knowledge/     # AIが参照する知識（仕様書・ルール・解析結果）
│   ├── game-features/   # ゲーム画面・機能仕様
│   └── masterdata/      # ID採番ルール・テーブル関係性
├── tasks/         # AIタスク定義（プロンプト・手順書）
└── tools/         # 実行可能ツール（Pythonスクリプト等）
```

新しい知識は `domain/knowledge/` に、再利用可能なAIタスクは `domain/tasks/` に追加する。

## Agent Skills (.claude/skills/)

Claude Code で利用可能な専用スキル（`/skill-name` で呼び出し、またはタスク説明から自動判定）:

| スキル | 用途 |
|--------|------|
| `masterdata-explorer` | DBスキーマ・CSV調査（jq/DuckDB） |
| `masterdata-csv-validator` | CSV投入可能性・スキーマ整合性チェック |
| `masterdata-ops-creator` | 運営設計書からマスタデータCSV一括生成 |
| `masterdata-ingame-creator` | インゲームマスタデータCSV生成 |
| `masterdata-ingame-verifier` | インゲームCSV品質検証 |
| `masterdata-releasekey-reporter` | リリースキー別データ抽出・レポート |
| `masterdata-id-numbering` | ID採番ルール確認・生成・検証 |
| `masterdata-csv-to-xlsx` | CSVをXLSX（Excel）に変換・統合 |
| `masterdata-table-docs-creator` | テーブル詳細解説ドキュメント生成 |
| `clickup-task-analyzer` | ClickUpタスクJSONからレポート生成 |
| `task-workflow-manager` | タスク管理・成果物積み上げワークフロー |

## マスタデータ作成ワークフロー（典型的な流れ）

1. `domain/knowledge/` でルール・仕様を確認
2. `masterdata-ops-creator` または `masterdata-ingame-creator` でCSV生成
3. `masterdata-csv-validator` でDB投入可能性を検証
4. `masterdata-releasekey-reporter` でリリースキー別レポート作成
