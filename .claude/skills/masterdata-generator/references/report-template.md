# REPORT.md テンプレート

**全てのマスタデータを生成した後に**、データ生成結果をMarkdown形式でレポートします。

## ファイルパス

```
マスタデータ/施策/[施策名]/REPORT.md
```

## テンプレート


## 重要な注意事項 ⚠️

### レポート作成のタイミング

- ✅ **レポートを書く前に**: 要件に含まれる全てのマスタデータが生成されていることを確認
- ❌ **絶対に作成しない**: 「未作成のマスタデータ」セクション

### 不明点がある場合の対処

途中で不明点がある場合は、既存データのパターンから推測して実装を続けてください。
レポートは全てのマスタデータが完成してから書きます。

### チェックリストの重要性

「データ整合性チェック」のチェックリストは、全ての項目を実際に確認した上でチェックを入れてください。
特に最後の項目「要件に含まれる全てのマスタデータが生成されている」は必須です。
