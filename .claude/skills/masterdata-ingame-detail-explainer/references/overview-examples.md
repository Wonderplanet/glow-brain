## event_dan1_challenge01_00001（イベントチャレンジ・砦破壊型）

**danシリーズのイベントチャレンジクエスト**（砦破壊型）。

- **砦HP**: `40,000`（ダメージ有効）
- **BGM**: 通常=`SSE_SBG_003_001`・ボス=`SSE_SBG_003_004`
- **コマ**: 3行構成（row1: 0.6+0.4、row2: 1.0、row3: 0.4+0.6、コマ効果なし）
- **グループ**: デフォルトのみ（ループなし・終端）
- **使用する敵**: 2種類（ボス1種: ターボババア / 雑魚1種: セルポ星人）
- **特殊ルール**: スピードアタックルールあり
- 砦HP99%以下（実質1ダメージ）でボス+雑魚5体が同時出現する砦ダメージトリガー設計

## raid_yuw1_00001（レイドバトル・スコアアタック型）

**yuwシリーズのレイドバトル**（スコアアタック型）。

- **砦HP**: `1,000,000`（ダメージ無効: is_damage_invalidation=1）
- **BGM**: 通常=`SSE_SBG_003_007`・ボスBGMなし
- **コマ**: 4行構成（row1: 1.0×1、row2: 0.5+0.5、row3: 1.0×1、row4: 0.25+0.5+0.25、コマ効果なし）
- **グループ**: デフォルト + w1〜w7（累計撃破数トリガー、w7→w1ループ）
- **使用する敵**: 12種類（ボス2種: 奥村正宗Boss×2色 / 中ボス6種 / 雑魚4種）
- **特殊ルール**: なし
- w3から1体差でw4・w5に並行分岐する設計（1ループ38体）
- 奥村正宗Bossのbase_HPが他ボスの5倍で全コンテンツ中最高クラスの実HP

## normal_dan_00002（メインクエスト・ノーマル難度）

**danシリーズのメインクエスト ノーマル難度**（砦破壊型）。

- **砦HP**: `15,000`（ダメージ有効）
- **BGM**: 通常=`SSE_SBG_003_001`・ボスBGMなし
- **コマ**: 3行構成（row1: 0.6+0.4[koma2に暗闇効果]、row2: 0.25+0.75[両コマ暗闇]、row3: 0.33+0.34+0.33[効果なし]、暗闇コマ合計3枚）
- **グループ**: デフォルト + group1（1体撃破でgroup1移行・終端）
- **使用する敵**: 2種類（雑魚2種: セルポ星人・ファントム）
- **特殊ルール**: なし
- デフォルトは暗闇コマ3枚クリア後に出現する初心者チュートリアル設計
