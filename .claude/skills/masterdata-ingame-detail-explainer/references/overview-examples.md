# 概要セクション 箇条書き記述例

## event_dan1_challenge01_00001（イベントチャレンジ・砦破壊型）

**danシリーズのイベントチャレンジクエスト**（砦破壊型）。

- **砦HP**: `40,000`（ダメージ有効）・背景アセット=`dan_0001`
- **BGM**: 通常=`SSE_SBG_003_001`・ボス=`SSE_SBG_003_004`
- **コマ**: 3行構成（row1: 0.6+0.4の2コマ、row2: 1.0の1コマ、row3: 0.4+0.6の2コマ、コマ効果なし）・アセット=`dan_00007`
- **グループ**: デフォルトのみ（ループなし・終端）
- **使用する敵**: 2種類（ボス1種: ターボババア・Blue / 雑魚1種: セルポ星人・Blue）
- **特殊ルール**: スピードアタックルールあり
- 開幕はコマ0・コマ1に雑魚3体を EnterTargetKoma 待ち伏せ配置。バトル開始2秒後に雑魚3体・10秒後に雑魚50体を大量連続投入。MstInGame 係数は全て 1.0 でシーケンス側でスケーリング
- 砦HP99%以下（実質1ダメージ）でボス+雑魚5体が同時出現する砦ダメージトリガー設計。ターボババアは非常に高速（雑魚の約2倍）・高耐久・高火力。雑魚セルポ星人は低速・ノックバックしやすい標準設定
- バトルヒントは黄属性推奨と早クリア報酬を明記。ステージ説明はチャレンジクエストの演出に沿った内容

## raid_yuw1_00001（レイドバトル・スコアアタック型）

**yuwシリーズのレイドバトル**（スコアアタック型）。

- **砦HP**: `1,000,000`（ダメージ無効: is_damage_invalidation=1）・背景アセット=`event_yuw_0002`
- **BGM**: 通常=`SSE_SBG_003_007`・ボスBGMなし
- **コマ**: 4行構成（row1: 1.0の1コマ=yuw_00003、row2: 0.5+0.5の2コマ、row3: 1.0の1コマ=yuw_00004、row4: 0.25+0.5+0.25の3コマ、全コマ効果なし）
- **グループ**: デフォルト + w1〜w7（累計撃破数トリガー `FriendUnitDead`・w7→w1ループ・1ループ38体）
- **使用する敵**: 12種類（ボス2種: 奥村正宗Boss×2色 / 中ボス6種: 天乃リリサ・橘美花莉・乃愛・喜咲アリア・753♡・奥村正宗Normal / 雑魚4種: ファントム赤・無属性）
- **特殊ルール**: なし
- デフォルトでファントム無属性11体を広域分散配置し1体撃破でw1起動。w3から1体差でw4・w5に並行分岐する設計
- 序盤ボスは強め・中盤は高耐久・w6〜w7は非常に高耐久と難度が段階上昇。奥村正宗BossのbaseHPが他ボスの5倍で全コンテンツ中最高クラスの実HP
- バトルヒントは高得点を狙う戦略（属性・キャラ選択）を中心に記載

## normal_dan_00002（メインクエスト・ノーマル難度）

**danシリーズのメインクエスト ノーマル難度**（砦破壊型）。

- **砦HP**: `15,000`（ダメージ有効）・背景アセット=`dan_0001`
- **BGM**: 通常=`SSE_SBG_003_001`・ボスBGMなし・ループ背景=`dan_00006`
- **コマ**: 3行構成（row1: 0.6+0.4の2コマ=koma2に暗闇効果、row2: 0.25+0.75の2コマ=両コマ暗闇、row3: 0.33+0.34+0.33の3コマ=効果なし、暗闇コマ合計3枚）
- **グループ**: デフォルト + group1（1体撃破でgroup1移行・終端）
- **使用する敵**: 2種類（雑魚2種: セルポ星人・Red（Defense型）/ ファントム・Red（Attack型））
- **特殊ルール**: なし
- デフォルトは暗闇コマ3枚クリア後に出現する初心者チュートリアル設計。セルポ星人は中速・ノックバックなし・控えめな強さ。ファントムは高速・ノックバック1回・hp補正が高くやや硬め
- group1 は終端グループで0.5秒・5秒・15秒のタイミングでセルポ星人を4体ずつ段階投入（全て控えめな強さで統一）。全敵が赤属性で青属性キャラが有利な構成
- バトルヒントは暗闇コマの効果と青属性有利を説明。初心者向けの暗闇コマ学習ステージとしての内容
