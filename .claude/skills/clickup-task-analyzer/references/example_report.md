## プロジェクト概要

- **バージョン**: v1.5.0

### 1. BOXガシャ（いいジャンくじ）

- イベントクエスト連携

- プリズム・広告視聴上限を999に拡張

- UI/UXの全面リニューアル

### 4. 特別ルール：作品バフ
- ステージスペシャルルール実装
- スキルシステム拡張

### 作業順序のパターン

各主要機能は以下の順序で実装：

1. **サーバー実装** → **クライアント実装** → **UI実装** → **統合・QA**

具体例（BOXガシャ）:
1. サーバーAPI実装 (2025-12-19完了)
2. クライアントData層 (2025-12-18完了)
3. クライアントDomain層 (2025-12-24完了)
4. UI画面実装 (2025-12-25～2026-01-05)
5. クライアント統合 (2026-01-13完了)
6. 全体統合 (2026-01-14完了)
