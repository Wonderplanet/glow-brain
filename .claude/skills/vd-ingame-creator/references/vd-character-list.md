## 共通ルール

### 通常敵（雑魚）
- 各作品のストーリーに登場するキャラをベースにする
- ノーマルブロックの登場雑魚は共通で **最低15体以上** 登場させること

### レアファントム
- **既存のファントムと同じデータを使用する**（名称・IDのみレアファントム用に調整）
- ステータス・能力は通常ファントムと共通

### ボスキャラ（強敵）
- 各作品のURキャラの対抗となるギミックやコマ効果を持つキャラを基本とする
- **オーラ**: 全作品共通で「降臨ボスオーラ1」を設定
- `aura_type` は `Boss`（MstAutoPlayerSequence）

## MstEnemyCharacter ID の確認クエリ

```bash
# 作品別のキャラ確認
duckdb -c "SELECT id, mst_series_id, asset_key FROM read_csv('projects/glow-masterdata/MstEnemyCharacter.csv', AUTO_DETECT=TRUE, nullstr='__NULL__') WHERE id LIKE '%{作品ID}%' ORDER BY id;"
```
