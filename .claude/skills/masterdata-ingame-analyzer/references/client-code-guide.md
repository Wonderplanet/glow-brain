# glow-client コード調査ガイド

## いつ参照するか

以下のケースで参照する（毎回必須ではない）:

- `action_type` / `condition_type` 等の enum 値の動作意味を確認したい
- HP計算・倍率の適用順序・スタック方法を理解したい
- コマ効果の発動タイミングや条件処理を確認したい
- NULL や空文字の扱い・デフォルト値を確認したい
- マスタデータの設計意図をクライアント側から読み取りたい
