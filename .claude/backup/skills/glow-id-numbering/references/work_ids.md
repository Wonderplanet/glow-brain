## 作品ID命名規則

- 作品名のローマ字表記の先頭3文字
- 小文字のみ使用
- 例: 「SPY×FAMILY」→ 「spy」、「ダンダダン」→ 「dan」

## 汎用作品ID

**`glo`** は GLOW ゲーム全体で共通利用するリソースに使用:
- 汎用アイテム: `prism_glo_00001`
- 汎用背景: `background_glo_00001`
- システム共通リソース

## 作品検索例

作品名から作品IDを見つける:
- 「スパイファミリー」「SPY×FAMILY」 → `spy`
- 「ダンダダン」 → `dan`
- 「ラーメン赤猫」 → `aka`

作品IDから作品名を見つける:
- `spy` → SPY×FAMILY
- `dan` → ダンダダン
- `chi` → チェンソーマン

## 注意事項

- 新規作品追加時は `作品.html` に登録してから各リソースIDを採番
- display_orderは表示順序を制御（001〜999）
- `glo`（999）は常に最後
