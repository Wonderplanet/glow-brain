# マスターテーブルリレーション一覧

このドキュメントは、GLOWプロジェクトの全マスターテーブルの列とそのリレーション関係をまとめたものです。

## 調査方法

- glow-serverのコード調査（列名パターン、マスタスキーマ）
- glow-clientのコード確認
- 列名パターンからの自動推測
- 個別のビジネスロジック調査

## リレーション表記について

- **なし**: リレーションなし（単体の値データ）
- **テーブル名.列名**: 他テーブルの指定列を参照
- **resource_typeに依存（動的）**: resource_typeの値に応じて参照先が変わる
- **なし（グループキー）**: 同テーブル内でのグルーピングキー
- **なし（主キー）**: テーブルの主キー
