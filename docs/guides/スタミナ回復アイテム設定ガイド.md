# スタミナ回復アイテム設定ガイド

## 📌 スタミナ回復アイテムとは

ゲーム内でプレイヤーのスタミナを回復できるアイテムです。
2種類の回復方式があり、用途に応じて使い分けます。

## 関連するテーブル

```
📊 MstItem（アイテム基本情報）← メイン
├─ アイテムの種類、回復量、期間などを設定
│
└─ 🔤 MstItemI18n（名前・説明文）
   └─ 日本語・英語・中国語の表示名を設定
```

## 2種類の回復アイテム

### 🍾 StaminaRecoveryPercent（パーセント回復）
スタミナ上限の◯％を回復するアイテム

**例：** 「スタミナボトル」- スタミナ上限の50%回復

### 🧃 StaminaRecoveryFixed（固定値回復）
スタミナを固定値で回復するアイテム

**例：** 「スタミナドリンク」- スタミナを10回復

## 設定例

### MstItem の設定

```csv
ENABLE,id,type,group_type,rarity,asset_key,effect_value,sort_order,start_date,end_date,release_key,item_type,destination_opr_product_id
e,stamina_item_glo_00001,StaminaRecoveryFixed,Etc,R,stamina_item_glo_00001,10,1,"2026-02-01 15:00:00","2037-12-31 23:59:59",202602010,StaminaRecoveryFixed,
```

**重要項目：**

- **id**: アイテム識別ID（例: `stamina_item_glo_00001`）
- **type**: 回復方式を指定
  - `StaminaRecoveryFixed` = 固定値回復
  - `StaminaRecoveryPercent` = パーセント回復
- **effect_value**: 回復量
  - 固定値の場合は回復するスタミナ数（例: `10` = 10回復）
  - パーセントの場合は上限の◯%（例: `50` = 50%回復）
- **rarity**: レアリティ（R、SR、SSR、UR）
- **start_date / end_date**: 使用可能期間

### MstItemI18n の設定

```csv
ENABLE,release_key,id,mst_item_id,language,name,description
e,202602010,stamina_item_glo_00001_ja,stamina_item_glo_00001,ja,スタミナドリンク,使用することでスタミナを最大10回復できるアイテム
e,202602010,stamina_item_glo_00001_en,stamina_item_glo_00001,en,Stamina Drink,An item that can recover up to 10 stamina
e,202602010,stamina_item_glo_00001_cn,stamina_item_glo_00001,zh,体力饮料,使用后可恢复最多10点体力的道具
```

**重要項目：**

- **mst_item_id**: MstItemのIDと一致させる
- **language**: `ja`（日本語）、`en`（英語）、`zh`（中国語）
- **name**: ゲーム内表示名
- **description**: アイテム説明文

## ✅ チェックポイント

1. **IDの重複確認** - 他のアイテムとIDが被っていないか確認
2. **回復量のバランス** - ゲーム内バランスとして適切な数値か確認
3. **多言語対応** - 日本語・英語・中国語の3言語すべて設定されているか確認

## 💡 次のステップ

アイテムを設定した後、以下の設定が必要になります：

- **MstShopItem** - ショップで販売する場合
- **MstGachaLineup** - ガチャから排出する場合
- **報酬テーブル** - クエストやミッションの報酬にする場合
