# GLOWゲーム コンテンツ一覧

このドキュメントは、GLOWゲームに実装されている全コンテンツを体系的にまとめたものです。
画面一覧（150画面）とマスターデータ（約165テーブル）から抽出・整理しています。

---

## 🎮 バトル・クエスト系

### 1. 通常クエスト
- ステージクリア型のバトルシステム
- クリア報酬、時間報酬あり
- ボーナスキャラクター設定
- 特別ルール設定可能
- **画面**: QuestSelect, QuestContentTop, InGame, BattleResult
- **マスターデータ**: MstQuest, MstStage, MstStageReward, MstQuestBonusUnit

### 2. 降臨バトル（AdventBattle）
- 期間限定の高難度バトル
- ランキング機能付き（個人ランキング + レイドランキング）
- ミッションシステム
- 専用報酬システム
- **画面**: AdventBattle, AdventBattleResult, AdventBattleRanking, AdventBattleRaidRankingResult, AdventBattleMission, AdventBattleRewardList
- **マスターデータ**: MstAdventBattle, MstAdventBattleRank, MstAdventBattleReward, MstAdventBattleClearReward

### 3. 強化クエスト（EnhanceQuest）
- キャラクター強化素材獲得用クエスト
- スコア評価システム
- **画面**: EnhanceQuestTop, EnhanceQuestScoreDetail
- **マスターデータ**: MstStageEnhanceRewardParam

### 4. イベントクエスト
- 期間限定ステージ
- イベントボーナスキャラクター設定
- イベント専用報酬
- イベント専用ルール
- **画面**: EventQuestTop, EventQuestSelect, EventBonusUnitList
- **マスターデータ**: MstEvent, MstEventBonusUnit, MstStageEventReward, MstStageEventRule, MstStageEventSetting

### 5. PvP/決闘
- プレイヤー対戦システム
- ランキングシステム
- シーズン制（シーズン開始・終了演出あり）
- 段位・ランク報酬
- マッチング機能
- 対戦相手詳細確認
- **画面**: PvpTop, PvpBattleResult, PvpBattleFinishAnimation, PvpRanking, PvpInfo, PvpOpponentDetail, PvpNewSeasonStart, PvpSeasonStart, PvpPreviousSeasonResult, PvpRewardList
- **マスターデータ**: MstPvp, MstPvpRank, MstPvpReward, MstPvpBonusPoint, MstPvpMatchingScoreRange, MstPvpDummy

### 6. キャラクターシステム

## 🎯 ゲームの特徴

GLOWは以下の要素を軸にした、コンテンツ豊富なモバイルゲームです：

1. **バトルシステム**: 多様なバトルモードと戦略性
2. **収集要素**: キャラクター、アートワーク、エンブレムなど多彩なコレクション
3. **育成システム**: レベル、グレード、ランクの3軸育成
4. **ミッション**: 6種類のミッションによる継続プレイ促進
5. **図鑑システム**: 包括的なコレクション記録
6. **マンガ演出**: 独自の漫画風バトル演出
7. **ソーシャル**: PvP、ランキング、コミュニティ機能
