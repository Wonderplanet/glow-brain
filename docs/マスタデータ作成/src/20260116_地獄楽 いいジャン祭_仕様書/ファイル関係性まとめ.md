# 地獄楽 いいジャン祭_仕様書 ファイル構造

## 概要
このディレクトリには、2026年1月16日から開催される「地獄楽 いいジャン祭」イベントの企画仕様書が格納されています。主に原作コミック1～4巻のストーリー再現をテーマとしたイベント設計書群で構成されています。

開催期間: 2026/01/16（金）15:00～2026/02/16（月）10:59

## 主要データの流れ

1. **企画概要の確定**: [01_概要.html] - イベント全体のコンセプト、新キャラ・新アイテム定義
2. **施策詳細設計**: [02_施策.html] - 各クエスト種別の設計
3. **ガシャ設計**: [06_ガシャ関連] - ピックアップガシャA・Bの確率設計とラインナップ
4. **報酬バランス調整**: [05_報酬一覧.html] - 全施策の報酬を集約してバランス調整
5. **ミッション設計**: [04_ミッション.html] - 達成条件と報酬の定義
6. **降臨バトル設計**: [03_降臨バトル.html] - ランキング報酬とバトル仕様
7. **ショップ設計**: [07_ショップ関連] - パック販売とプリズム設計
8. **クリエイティブ依頼**: [地獄楽 クリエイティブ依頼.html] - バナー、背景、アセットの発注
9. **スケジュール確定**: [告知スケジュールNEO.html] - 告知・実装スケジュールの最終調整

## 注意事項

### 命名規則
- 番号付きファイル（00～11）は処理順序に対応
- 「のコピー」「のテンプレ」はバックアップまたはテンプレート
- 「_REBORN」は代替バージョン
- 「使用しない_」は廃棄予定

### ガシャラインナップの違い
- **ガシャA**: 「賊王 亜左 弔兵衛」「山田浅ェ門 桐馬」がピックアップ、「がらんの画眉丸」をラインナップから除外
- **ガシャB**: 「がらんの画眉丸」「山田浅ェ門 桐馬」がピックアップ

### 主要スケジュール
- デイリークエスト: 2026/01/16 15:00 ～ 2026/02/02 3:59
- ストーリークエスト（画眉丸編）: 2026/01/16 15:00 ～ 2026/02/16 10:59
- ストーリークエスト（共闘関係編）: 2026/01/21 15:00 ～ 2026/02/16 10:59
- 降臨バトル: 2026/01/23 15:00 ～ 2026/01/29 14:59

### マスタデータ作成時の注意
1. 新キャラIDは「chara_jig_」、敵専用は「enemy_jig_」で始まる
2. かけらIDは「piece_jig_」、メモリーは「memory_chara_jig_」
3. エンブレムは「emblem_event_jig_」または「emblem_adventbattle_jig_」
4. 原画は「artwork_event_jig_」で管理
5. すべてのIDはGLOW_ID管理シートに登録済み
