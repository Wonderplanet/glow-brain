# 100カノいいジャン祭 仕様書ファイル構造

## 概要
このディレクトリには「君のことが大大大大大好きな100人の彼女 いいジャン祭」のイベント仕様書が含まれています。
元はGoogleスプレッドシートで、HTMLファイルとして出力されたものです。

## ファイル構造

### 1. メイン目次
- **企画仕様書_目次.html**
  - 全仕様書の目次・インデックス
  - 各セクションへのリンクと構成が記載

### 2. コア仕様書ファイル（番号順）
以下のファイルが企画の主要仕様を構成:

#### 00番台: 準備資料
- **00_ロードマップ転記用.html** - プロジェクト全体のロードマップ情報

#### 01番台: 企画概要
- **01_概要.html** - イベント全体の概要・基本情報
- **01_概要 のテンプレ.html** - 他イベント用のテンプレート

#### 02番台: 施策詳細
- **02_施策.html** - メイン施策（クエスト関連）の詳細仕様
- **02_施策_REBORN.html** - 施策の改訂版
- **02_施策テンプレ のコピー.html** - テンプレート

#### 03番台: バトル・制作依頼
- **03_降臨バトル.html** - 降臨バトルシステムの仕様
- **03_制作依頼一覧.html** - 必要な制作物のリスト
- **03_制作＆監修依頼一覧.html** - 制作・監修依頼の統合リスト
- **03_制作依頼一覧 のテンプレ.html** - テンプレート
- **使用しない_制作＆監修依頼一覧.html** - 未使用ファイル

#### 04番台: ミッション・監修
- **04_ミッション.html** - イベントミッション仕様（強敵撃破、クエストクリアなど）
- **04_監修に依頼したいもの.html** - 監修依頼事項
- **04_監修に依頼したいもの テンプレ.html** - テンプレート

#### 05番台: 報酬
- **05_報酬一覧.html** - イベント全体の報酬集計（ログボ、デイリー、ストーリー、チャレンジ、ミッション、降臨バトルなど）

#### 06番台: ガシャ
- **06_ガシャ目次.html** - ガシャ関連ファイルの目次
- **06_ガシャ基本仕様.html** - ガシャの基本確率・仕様
- **06_ピックアップガシャA_注意事項.html** - ピックアップガシャAの注意事項
- **06_ピックアップガシャB_注意事項.html** - ピックアップガシャBの注意事項
- **君のことが大大大大大好きな100人の彼女 いいジャン祭ピックアップガシャA_設計書.html** - ガシャA詳細設計
- **君のことが大大大大大好きな100人の彼女 いいジャン祭ピックアップガシャB_設計書.html** - ガシャB詳細設計

#### 07番台: ショップ
- **07_ショップ目次.html** - ショップ関連ファイルの目次
- **07_ショップ_要件書.html** - ショップの要件定義
- **07_いいジャン祭パック_設計書.html** - 有料パックの設計書

### 3. クリエイティブ・アセット関連
- **君のことが大大大大大好きな100人の彼女 クリエイティブ依頼.html** - クリエイティブ制作依頼まとめ
- **バナー一覧.html** - 必要なバナーのリスト
- **バナー作成依頼.html** - バナー制作依頼書
- **背景一覧.html** - 使用背景のリスト
- **アセット一覧.html** - 必要アセットのリスト
- **クリエイティブ一覧.html** - クリエイティブ素材の総合リスト

### 4. スケジュール関連
- **告知スケジュール のコピー.html** - 告知スケジュールの複製
- **告知スケジュールNEO.html** - 更新版の告知スケジュール

### 5. その他補助ファイル
- **100カノ交換所.html** - イベント交換所の仕様
- **memo用リソース計算用シート.html** - リソース計算用のメモシート

├── 04_ミッション.html            │
└── 05_報酬一覧.html ─────────────┘
    ├── 06_ガシャ目次.html ───────┐

## 主要データの流れ

1. **企画立案**: 01_概要.html で全体像を定義
2. **施策設計**: 02_施策.html、03_降臨バトル.html でゲーム内容を設計
3. **報酬設計**: 04_ミッション.html、05_報酬一覧.html で報酬バランスを調整
4. **収益化**: 06_ガシャ、07_ショップ で課金要素を設計
5. **制作依頼**: クリエイティブ関連ファイルで必要素材を洗い出し
6. **スケジュール**: 告知スケジュールで公開タイミングを管理

## テンプレートファイル
以下は他のイベントでも使用可能なテンプレート:
- 01_概要 のテンプレ.html
- 02_施策テンプレ のコピー.html
- 03_制作依頼一覧 のテンプレ.html
- 04_監修に依頼したいもの テンプレ.html

## 注意事項
- ファイル名に"テンプレ"や"コピー"が含まれるものは参考用
- "使用しない_"で始まるファイルは廃棄予定
- 正式版は番号付きファイル(01_、02_など)を参照すること
