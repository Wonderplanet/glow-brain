## 使用方法

### データの確認
各CSVファイルは既存のマスタデータと同じ形式で作成されています。

### インポート手順
1. 各CSVファイルの内容を確認
2. リリースキー（202601010）を実際のリリース予定に合わせて調整
3. IDの重複がないことを確認
4. データベースにインポート

### 注意事項
- `release_key` は仮の値（202601010）を設定しています
- キャラクターID（chara_you_*）は仮のIDです
- アイテムID（memory_glo_*, gacha_ticket_*）は既存IDを使用しています
- 実際のリリース前に各IDを確認・調整してください
