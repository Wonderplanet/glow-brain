# シート分類

スプレッドシート「【ストーリー】必ず生きて帰る」の全25シートをGASとの関係で分類したものです。

---

## GASのシステムシート（ツールが使う設定・出力用）

| シート名 | 役割 |
|---------|------|
| `チェック対象管理` | チェック対象の設計書シートとマスターURLを管理 |
| `master_config` | 照合先マスタースプシの読み取り開始位置を定義 |
| `マッピング設定シート_new` | 設計書→マスターのマッピングを定義 |
| `MstAutoPlayerSequence` | GASが設計書から生成した中間シート（出力結果） |

---

## チェック対象の設計書シート（チェック対象管理に登録済み）

| シート名 | チェック列 | 状態 |
|---------|----------|------|
| `1話` | FALSE | 登録済みだが現在は実行されない |
| `2話` | FALSE | 同上 |
| `3話` | FALSE | 同上 |
| `4話` | FALSE | 同上 |
| `5話` | FALSE | 同上 |
| `6話` | **TRUE** | **現在アクティブ（実行される）** |

- 全シート共通で `マッピング設定シート_new` が使われ、`MstAutoPlayerSequence` に出力される
- `1話〜5話` はチェック列を `TRUE` にすればすぐに対象に追加できる

## マッピングの概要

`マッピング設定シート_new` は `MstAutoPlayerSequence` の1テーブルのみに対応しています。

| 項目 | 内容 |
|------|------|
| 出力マスター | `MstAutoPlayerSequence` のみ |
| 総列数 | 34列 |
| ユニークキー | `sequence_element_id` |
| 下方継承の列 | `sequence_set_id`（敵ゲートID）、`release_key`（リリースキー） |
| チェック無視の列 | `id`、`last_boss_trigger` |
