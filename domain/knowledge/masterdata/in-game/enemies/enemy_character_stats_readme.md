# enemy_character_stats.csv 列定義

全敵キャラクター（144体）のMstEnemyStageParameterを集計した統計CSV。
各キャラが持つ全バリアントの平均・最小・最大値と、シリーズ内ランク・全作品パーセンタイルを収録。

---

## 基本情報列

| 列名 | 説明 |
|------|------|
| `enemy_character_id` | `MstEnemyCharacter.id` |
| `name_ja` | キャラクター名（日本語） |
| `series` | 作品ID（`mst_series_id`） |
| `is_phantomized` | 幻体フラグ。`1` = プレイアブルキャラの幻体（敵として登場）、`0` = 専用の敵キャラ |
| `variant_count` | このキャラの `MstEnemyStageParameter` レコード数（バリアント数） |
| `series_chars_total` | 同シリーズの総キャラ数（ランクの分母） |
| `global_chars_total` | 全作品の総キャラ数（パーセンタイルの母数） |

---

## ステータス列（HP / 攻撃力 / 移動速度 / ウェル距離 / バトルポイント）

各ステータスについて以下の列が存在する。

| サフィックス | 説明 |
|------------|------|
| `{stat}_avg` | 全バリアントの平均値 |
| `{stat}_min` | 全バリアントの最小値 |
| `{stat}_max` | 全バリアントの最大値 |
| `{stat}_rank_in_series` | **シリーズ内ランク**（1 = そのシリーズで最も高い値） |
| `{stat}_pct_global` | **全作品パーセンタイル**（0〜100。100 = 全キャラ中トップ水準） |

### 対象ステータス

| プレフィックス | ステータス名 | 高いほど | 備考 |
|------------|------------|---------|------|
| `hp` | HP | 耐久が高い | — |
| `atk` | 攻撃力 (`attack_power`) | 攻撃が強い | — |
| `spd` | 移動速度 (`move_speed`) | 素早い | — |
| `wd` | ウェル距離 (`well_distance`) | ゴールに近い位置まで侵入する | **低いほど危険**。パーセンタイルは「低い値 = 高いパーセンタイル」として計算 |
| `bp` | バトルポイント報酬 (`drop_battle_point`) | 倒したときの報酬が大きい | — |

---

## 読み方の例

```
enemy_dan_00201（ターボババア）の行:
  spd_avg = 62.5
  spd_rank_in_series = 1   → danシリーズ11体中 最速
  spd_pct_global = 99.3    → 全作品144体中 上位0.7%のスピード

enemy_dan_00001（セルポ星人）の行:
  hp_avg = 10773
  hp_rank_in_series = 10   → danシリーズ11体中 10番目（ほぼ最低）
  hp_pct_global = 35.0     → 全作品で下位35%の耐久
```

---

## 集計条件

- `MstEnemyStageParameter.ENABLE = 'e'` のレコードのみ対象
- `is_phantomized` の別なく全キャラを同一母数で比較（シリーズ内ランク・全作品パーセンタイルともに）
- シリーズ内ランクは同シリーズの全キャラ（幻体含む）との比較
