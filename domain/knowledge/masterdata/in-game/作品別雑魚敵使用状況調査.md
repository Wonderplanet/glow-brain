# 作品別 雑魚敵（MstEnemyCharacter）使用状況調査

作成日: 2026-03-01

## 調査概要

各作品（MstSeries）のインゲームステージで、どの雑魚敵（MstEnemyCharacter.id）が多く使われているかを調査した。

### 調査方法

以下のテーブルをJOINして、`character_unit_kind = 'Normal'`（雑魚敵）かつ本番データ（`release_key < 999999999`）のみを対象に集計。

```
MstAutoPlayerSequence（action_type='SummonEnemy', action_value=MstEnemyStageParameter.id）
  ↓ JOIN
MstEnemyStageParameter（mst_enemy_character_id = MstEnemyCharacter.id）
  ↓ JOIN
MstEnemyCharacter（mst_series_id = 作品コード）
```

### 注意事項

- `character_unit_kind = 'Normal'` のもののみを雑魚敵として集計（Bossは除外）
- テスト用データ（release_key=999999999）は除外
- `glo`・`aya` はMstSeriesに存在しない（チュートリアル・内部テスト用の可能性）

## 傾向分析

### 雑魚敵の選定パターン

| パターン | 特徴 | 該当作品 |
|---------|------|---------|
| **1体主力型（圧倒的）** | 1体が80%以上を占める | chi, sur, rik, mag（mag_00301） |
| **1体主力型** | 1体が50〜80%、残りはサブ | spy, kai, sum, aka, kim |
| **2体体制型** | 上位2体が拮抗（5:3〜6:4程度） | dan, you |
| **多体分散型** | 3体以上を均等〜分散して使用 | jig, gom, yuw, osh, hut |

### enemy_* vs chara_* の使い分け

- `enemy_*` プレフィックス：専用の敵キャラ（そのシリーズの悪役・怪物など）
- `chara_*` プレフィックス：作品のキャラクターを敵として使用

多くの作品で `enemy_*` を主力にしているが、`osh`・`yuw`・`rik`・`hut` など一部の作品は `chara_*` が主力。これは作品の設定上「悪役モンスター」が少ない作品では、作品キャラクターを敵として登用していると推測される。

## DuckDBクエリ（再現用）

```sql
SELECT
  ec.mst_series_id as series_id,
  ec.id as enemy_character_id,
  COUNT(*) as usage_count
FROM read_csv('projects/glow-masterdata/MstAutoPlayerSequence.csv', AUTO_DETECT=TRUE, nullstr='__NULL__') seq
JOIN read_csv('projects/glow-masterdata/MstEnemyStageParameter.csv', AUTO_DETECT=TRUE, nullstr='__NULL__') esp
  ON seq.action_value = esp.id
JOIN read_csv('projects/glow-masterdata/MstEnemyCharacter.csv', AUTO_DETECT=TRUE, nullstr='__NULL__') ec
  ON esp.mst_enemy_character_id = ec.id
WHERE seq.action_type = 'SummonEnemy'
  AND seq.ENABLE = 'e'
  AND seq.release_key < 999999999
  AND esp.ENABLE = 'e'
  AND ec.ENABLE = 'e'
  AND esp.character_unit_kind = 'Normal'
GROUP BY ec.mst_series_id, ec.id
ORDER BY ec.mst_series_id, usage_count DESC;
```
