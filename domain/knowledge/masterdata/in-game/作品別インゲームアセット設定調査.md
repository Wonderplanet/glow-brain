# インゲームマスタデータ設定 - 作品別アセット調査報告書

**調査日**：2026年3月1日
**調査目的**：作品（シリーズ）ごとのアセットが、インゲームのコンテンツタイプや各データごとに違いがあるかどうかを検証
**データソース**：`projects/glow-masterdata/MstInGame.csv` 他（v1.6.1ブランチ、560行）

## 2. コンテンツ種別の分類

### 2.1 通常ステージ（Normal/Hard/VeryHard）
- **対象作品**：spy, dan, gom, mag, kai, jig, glo, aka, chi, rik, tak, sum, dos, aha, ron, bat（16作品）
- **パターン**：`normal_XXX`, `hard_XXX`, `veryhard_XXX`（各作品6ブロック×3難易度 = 18ブロック/作品）
- **合計**：288ブロック

### 2.2 イベントステージ（Event）
- **パターン**：`event_XXX1_charaget01`, `event_XXX1_charaget02`, `event_XXX1_1day` 等
- **対象作品**：spy, dan, mag, kai, yuw, sur, osh, jig, you, kim, hut, f, l, aya
- **1作品あたり平均**：20〜26ステージ

### 2.3 降臨バトル（Raid）
- **パターン**：`raid_XXX1_00001`
- **特性**：ボスBGM（boss_bgm_asset_key）が設定される場合あり
- **対象作品**：spy, dan, mag, yuw, sur, osh, jig, you, kim, hut, f, l, aya

### 2.4 ランクマッチ
- **パターン**：`pvp_XXX_01`, `pvp_XXX_02`
- **特性**：BGMが全作品統一（SSE_SBG_003_007）
- **対象**：全シリーズ

### 2.5 その他
- **Tutorial**：tutorial_1, tutorial_2, tutorial_3
- **Develop/Test**：develop_001, plan_test_stage_XXX 等
- **Enhance**：enhance_00001

### 3.1 BGMアセットキー（bgm_asset_key）

| 作品 | 通常ステージ | イベント | レイド | パターン |
|------|-----------|---------|-------|---------|
| **spy** | SSE_SBG_003_002 | SSE_SBG_003_002 | SSE_SBG_003_007 | 異なる |
| **dan** | SSE_SBG_003_001 | SSE_SBG_003_001 | SSE_SBG_003_007 | 異なる |
| **gom** | SSE_SBG_003_006 | - | - | 独自 |
| **mag** | SSE_SBG_003_001 | SSE_SBG_003_001 | SSE_SBG_003_007 | 異なる |
| **kai** | SSE_SBG_003_001 | SSE_SBG_003_001/002 | - | 混在 |
| **jig** | SSE_SBG_003_003 | - | SSE_SBG_003_008 | 異なる |
| **glo** | SSE_SBG_003_008 | SSE_SBG_003_002 | - | 混在 |
| **aka** | SSE_SBG_003_006 | - | - | 共有 |
| **yuw** | - | SSE_SBG_003_006 | SSE_SBG_003_007 | 異なる |
| **sur** | - | SSE_SBG_003_001 | SSE_SBG_003_007 | 異なる |
| **osh** | - | - | SSE_SBG_003_006 | 独自 |

**重要なポイント**：
- 作品ごとにBGMアセットが異なる
- 通常ステージ用には8種類のBGMアセットが存在（SSE_SBG_003_001〜009）
- レイド/ダンジョンは汎用BGM（SSE_SBG_003_007が多い）を使用
- イベントステージは作品別BGMを流用することが多い

### 3.2 背景アセット（loop_background_asset_key）

| 作品 | 背景アセットキー例 | パターン |
|------|-----------------|--------|
| **spy** | spy_00005, spy_00002, spy_00001 | 作品専用（spy_XXXXX） |
| **dan** | dan_00005, dan_00006, dan_00007 | 作品専用（dan_XXXXX） |
| **gom** | gom_00001, gom_00002 | 作品専用（gom_XXXXX） |
| **mag** | mag_00004, mag_00006 | 作品専用（mag_XXXXX） |
| **kai** | glo_00010, kai_00001 | 混在（共有+作品専用） |
| **jig** | jig_00003, jig_00002 | 作品専用（jig_XXXXX） |
| **glo** | glo_00001, glo_00002, glo_00022 | 共有（glo_XXXXX） |
| **aka** | glo_00001 | 共有（gloアセット使用） |

**重要なポイント**：
- 各作品は専用の背景アセットを保有（`作品コード_XXXXX` 形式）
- GLOシリーズ（glo_XXXXX）はプロジェクト全体で共有される
- 一部の作品（kai、aka等）は共有アセットとの混在を使用
- **同一作品内でもステージごとに異なる背景アセットが指定される**

### 3.3 ボスBGMアセット（boss_bgm_asset_key）

| アセットキー | 主な用途 |
|-----------|--------|
| `SSE_SBG_003_004` | イベント/チャレンジのCharacter取得ステージ |
| `SSE_SBG_003_006` | 特定作品（osh等）のボスBGM |

**設定パターン**：
- 通常/ハード/ベリーハード：ボスBGM未設定（空欄）
- イベント（charaget）：ボスBGM設定あり（SSE_SBG_003_004）
- レイド：ボスBGMなし

### 3.4 プレイヤー砦アセット（player_outpost_asset_key）

|------|-----------|------|

| **kai（一部）** | kai_00001 | 作品専用 |
| **その他ほぼ全作品** | 空欄 | 未設定 |

## 6. 特別ルール設定（MstInGameSpecialRule）

イベント/チャレンジステージでは作品・難易度ごとに特別ルールが設定される。

| 作品 | 特別ルール | 内容 |
|------|-----------|------|
| **hut** | quest_event_hut1_Hp/AttackPower | 属性別HP+20%, 攻撃力+20% |
| **spy, kai, sur 等** | PartySeries 指定 | 特定シリーズキャラのみパーティ組成可 |
| **複数作品** | PartyRoleType | Role別（Attack/Defense等）の制限 |

## 7. テーブル間の依存関係

```
MstInGame
  ├─ mst_auto_player_sequence_set_id → MstAutoPlayerSequence（敵出現シーケンス）
  ├─ mst_page_id → MstPage（ページ構成）
  ├─ mst_enemy_outpost_id → MstEnemyOutpost（敵砦）
  ├─ boss_mst_enemy_stage_parameter_id → MstEnemyStageParameter（ボスパラメータ）
  └─ (間接) → MstInGameSpecialRule（特別ルール）
```

アセット値（bgm_asset_key, loop_background_asset_key等）は **MstInGameテーブル内に直接格納**（他テーブル参照なし）。

## 8. 結論

### 作品ごとのアセットは明確に異なる

| アセット種別 | 作品間の違い | コンテンツ間の違い |
|-----------|-----------|---------------|
| **BGMアセット** | ✅ 異なる（グループ分類あり） | ✅ 異なる（通常=作品別、Raid/ランクマッチ=統一） |
| **背景アセット** | ✅ 異なる（作品専用IDを保有） | ✅ 異なる（通常=複数パターン、Event/Raid=なし） |
| **ボスBGM** | △ 共通（SSE_SBG_003_004が多数） | ✅ 異なる（通常=なし、charaget=あり） |
| **プレイヤー砦** | ✅ 異なる（ほぼ未設定） | △ コンテンツ問わず未設定が大半 |

### 新作品追加時の考慮事項

1. **BGMアセットキーの決定**：既存グループ（A〜E）を参考に選択
2. **背景アセット準備**：通常ステージ用に最低6〜18種類（3難易度×6ブロック）
3. **特別ルール設定**：作品・難易度ごとのRole/Series制限を定義
4. **イベント時**：charaget ステージには boss_bgm_asset_key（SSE_SBG_003_004）を設定
