# MstEventDisplayReward 詳細説明

> CSVパス: `projects/glow-masterdata/MstEventDisplayReward.csv`

---

## 概要

`MstEventDisplayReward` は**イベントクエスト画面で表示する目玉報酬を定義するテーブル**。

DBスキーマのコメントに「(現在未使用)旧イベントクエスト画面で出す目玉報酬の設定」と明記されている通り、このテーブルは**現在使用されていない（廃止済み）テーブル**。旧イベントUI設計時に作られた仕様で、リニューアル後の現行UIでは参照されていない。

CSVの行数は0件（データは一切存在しない）。

### 経緯

旧イベントクエスト画面ではプレイヤーへの訴求のために「このイベントで獲得できる報酬」を画面上部に表示する設計だった。このテーブルはその表示用データを格納するために作られたが、UI設計の変更に伴い使用されなくなった。現行のイベント報酬表示はミッション・ガチャなど個別のコンテンツ画面で行われている。

### 関連テーブルとの構造図

```
MstEvent（イベント基本設定）
  └─ id → MstEventDisplayReward.mst_event_id（現在未使用）
```

## 設定時のポイント

1. **このテーブルは現在未使用であり、新規データの追加は不要**。DBスキーマのコメントで「現在未使用」と明示されている廃止済みテーブル。

2. **既存の空CSVファイルは互換性のために維持する**。サーバーのCSV読み込み処理が存在する場合、ファイル自体を削除するとエラーになる可能性があるため、空のまま保持する。

3. **新機能の開発で目玉報酬表示が必要になった場合は新しいテーブルを検討する**。旧仕様の `resource_type` が `varchar` 型（enumではない）で設計されているなど、現行の設計方針と一致しない部分があるため、再活用は推奨しない。

4. **クライアントの `MstEventDisplayRewardData.cs` クラスは定義されている**が、実際のデータが存在しないため参照されることはない。クライアント側の実装が残っていても動作に影響しない。
