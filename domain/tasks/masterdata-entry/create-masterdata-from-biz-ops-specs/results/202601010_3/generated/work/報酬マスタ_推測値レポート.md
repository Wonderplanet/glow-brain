## 4. sort_orderについて

### MstMissionReward.sort_order
- **値**: すべて `1`
- **理由**: 設計書に表示順序の記載がなかったため、デフォルト値`1`を設定
- **確認事項**: 同一グループ内で複数の報酬がある場合、表示順序を調整してください

### MstStageEventReward.sort_order
- **値**: 複数報酬がある場合は連番(1, 2, 3, ...)
- **理由**: 設計書に表示順序の記載がなかったため、登録順に連番を設定
- **確認事項**: 報酬の表示順序が正しいことを確認してください
