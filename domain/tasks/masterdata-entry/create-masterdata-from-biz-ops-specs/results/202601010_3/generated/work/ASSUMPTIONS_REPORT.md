## 次のステップ

### 1. 詳細なヒーロー設計書の入手
以下の情報を含む詳細設計書が必要です:
- 正確なステータス値 (HP、攻撃力、コスト等)
- アビリティの詳細定義 (種類、パラメータ、解放ランク)
- 攻撃アクションの詳細 (フレーム数、威力、エフェクト等)
- MstAttackElementの152カラム設定
- 必殺技名、セリフ、フレーバーテキスト

### 2. マスタデータの検証
生成したマスタデータCSVを以下の方法で検証してください:
```bash
python .claude/skills/masterdata-csv-validator/scripts/validate_all.py \
  --csv domain/tasks/masterdata-entry/create-masterdata-from-biz-ops-specs/results/202601010_3/generated/
```

### 3. ゲームバランス調整
ゲームデザイナーと協力して、以下を調整してください:
- ステータスバランス (HP、攻撃力、コスト等)
- アビリティバランス (効果値、発動条件等)
- キャラクター間の相対的な強さ

### 4. 公式設定との照合
原作漫画「地獄楽」の設定やマーケティング資料と照合し、以下を確認してください:
- キャラクター名、フレーバーテキスト
- 必殺技名、セリフ
- 戦闘スタイル、役割
