## 敵キャラ設計

### 通常敵（雑魚）

- 各作品のストーリーに登場するキャラをベースにする
- メインクエストnormal難易度（ストーリークエスト）のステータスをベースに設定

### レアファントム

- **既存のファントムと同じデータを使用する**（名称・IDのみレアファントム用に調整）
- ステータス・能力は通常ファントムと共通
- ID命名：レアファントム専用のID体系に従って付与

### ボスキャラ（強敵）

- 各作品のURキャラの対抗となるギミックやコマ効果を持つキャラを基本とする
- **オーラ**: 全作品共通で「降臨ボスオーラ1」を設定
- ステータスはフロア係数 1.00 時の強敵ステータスをベースに設定

## MstEnemyStageParameter ID採番ルール

> 降臨バトルのID採番ルールをベースに、VD（限界チャレンジ）向けに定義したルールです。
> 参考: `domain/tasks/20260310_115400_vd_ingame_masterdata_generation/knowledge/MstEnemyStageParameter_降臨バトルID採番ルール.md`

### IDフォーマット

```
{prefix}_{作品ID}_{キャラID}_vd_{ブロック種別}_{属性}
```

### 各セグメント

| セグメント | 内容 | 値の例 |
|-----------|------|--------|
| **prefix** | キャラの種別区分 | `c`（プレイヤーキャラが敵登場）/ `e`（敵専用キャラ） |
| **作品ID** | 作品のローマ字先頭3文字（小文字） | `spy`, `dan`, `gom`, `kai` |
| **キャラID** | 5桁の連番（MstEnemyCharacterと対応） | `00001`, `00101`, `00201` |
| **vd** | 限界チャレンジを示す固定文字列 | `vd`（必ず含める） |
| **ブロック種別** | 登場するブロックの種別 | `Boss`, `Normal`, `Rare` |
| **属性** | カラー | `Red`, `Blue`, `Green`, `Yellow`, `Colorless` |

### ブロック種別と用途

| ブロック種別 | 用途 |
|------------|------|
| `Normal` | ノーマルブロックの雑魚敵 |
| `Boss` | ボスブロックの強敵（ボス） |
| `Rare` | ノーマルブロックのレアファントム（レア敵） |

### prefix の決定ルール

| prefix | mst_enemy_character_id の接頭語 | 意味 |
|--------|-------------------------------|------|
| `c_` | `chara_xxx_xxxxx` | プレイヤーキャラクターが敵として登場 |
| `e_` | `enemy_xxx_xxxxx` | VD専用の敵キャラ |

### ID例

```
c_spy_00501_vd_Boss_Red        # SPY×FAMILYのキャラ ボスブロック 赤属性
c_dan_00001_vd_Normal_Colorless # ダンダダンのキャラ ノーマルブロック 無属性
e_spy_00001_vd_Rare_Colorless  # SPY×FAMILY専用敵 レアファントム 無属性
```

## ノーマルブロック抽選仕様

1. ボスブロックに辿り着くまでに **3パターンのノーマルブロックの組み合わせ** を抽選
2. 次に、その組み合わせに **どの作品のブロックを当てはめるか** を抽選
3. 最後に **どの作品のボスブロックを当てはめるか** を抽選し、1シーケンスが完成
