# 限界チャレンジ（vd）normalブロック マスタデータ要件

限界チャレンジの **normalブロック（`vd_normal`）** に特化したインゲームマスタデータ作成要件ドキュメントです。

---

## 概要

- **コンテンツ種別**: `vd`
- **ブロック種別**: `Normal`
- **IDプレフィックス**: `vd_{作品ID}_normal_{連番5桁}`
- **構成単位**: `MstInGame.id` 1件 = 1ブロック

### 動作仕様

フロア毎に配置された敵が、味方キャラが該当フロアに進行した際に起動する **受動的な構成**。

---

## 固定値

| 項目 | 値 | 備考 |
|------|---:|------|
| `MstEnemyOutpost` HP | `100` | 固定値（変更不可） |
| `MstKomaLine` 行パターン | **12パターンから1つ** | マスタデータ生成時にランダム選択（`コマ設計_行パターン.csv` 参照） |
| `MstEnemyOutpost.is_damage_invalidation` | 空 | ダメージ有効 |

---

## ID命名規則

```
vd_{作品ID}_normal_{連番5桁}
```

**例**: `vd_kai_normal_00001`

- `{作品ID}` はキャラシリーズ略称（`kai` / `dan` / `spy` 等）
- MstInGame.id・MstAutoPlayerSequence.sequence_set_id・MstPage.id・MstEnemyOutpost.id はすべて同一値を使用する

### MstEnemyStageParameter.id の短縮形

| ケース | ID短縮形 |
|--------|---------|
| vd（boss/normalで共通） | `{作品ID}_vd` |

**例**: `e_kai_00001_kai_vd_Normal_Colorless`

## 生成対象テーブル

| テーブル | 必須度 | 内容 |
|---------|--------|------|
| `MstEnemyStageParameter` | **必須** | 敵パラメータ（雑魚分） |
| `MstEnemyOutpost` | **必須** | HP = 100（固定） |
| `MstPage` | **必須** | バトルフィールドのページID |
| `MstKomaLine` | **必須** | コマライン（行パターンリストからランダム選択） |
| `MstAutoPlayerSequence` | **必須** | 敵出現シーケンス |
| `MstInGame` | **必須** | インゲーム全体設定 |

---

## MstAutoPlayerSequence シーケンス設計

### 行数目安

`3 〜 8行`

### シーケンス設定ルール

- **フェーズ切り替えはしない**（`SwitchSequenceGroup` は使用しない）
- `action_type` は `SummonEnemy` のみで構成する
- グループ（`sequence_group_id`）は空（デフォルトグループ）のみで設計する

### 基本シーケンスパターン

```
行1: ElapsedTime(250) → SummonEnemy(雑魚A) × N体
行2: ElapsedTime(1500) → SummonEnemy(雑魚B) × N体
行3: ElapsedTime(3000) → SummonEnemy(雑魚A) × N体  ※任意
```

- 雑魚を時間差で出現させる構成
- `aura_type`: すべて `Default`
- フェーズ切り替えはしないため `SwitchSequenceGroup` は不使用

## コマ設計（MstKomaLine）

**行パターン：マスタデータ生成時に `コマ設計_行パターン.csv` の12パターンの中から1つをランダム選択**:

| 行パターンID | コマ数 | パターン | コマ幅1 | コマ幅2 | コマ幅3 | コマ幅4 |
|-------------|--------|---------|--------|--------|--------|--------|
| 1 | 1コマ | 1コマのみ | 1.00 | | | |
| 2 | 2コマ | 右ちょい長2コマ | 0.60 | 0.40 | | |
| 3 | 2コマ | 左ちょい長2コマ | 0.40 | 0.60 | | |
| 4 | 2コマ | がっつり右長2コマ | 0.75 | 0.25 | | |
| 5 | 2コマ | がっつり左長2コマ | 0.25 | 0.75 | | |
| 6 | 2コマ | 2等分 | 0.50 | 0.50 | | |
| 7 | 3コマ | 3等分 | 0.33 | 0.34 | 0.33 | |
| 8 | 3コマ | 右広い | 0.50 | 0.25 | 0.25 | |
| 9 | 3コマ | 中央広い | 0.25 | 0.50 | 0.25 | |
| 10 | 3コマ | 左広い | 0.25 | 0.25 | 0.50 | |
| 11 | 3コマ | 中央狭い | 0.40 | 0.20 | 0.40 | |
| 12 | 4コマ | 4等分 | 0.25 | 0.25 | 0.25 | 0.25 |

```
例（行パターンID=7「3等分」を選択した場合）:
row=1: height=1.0, koma1_width=0.33, koma2_width=0.34, koma3_width=0.33

例（行パターンID=6「2等分」を選択した場合）:
row=1: height=1.0, koma1_width=0.50, koma2_width=0.50
```

- 行パターンはブロックごとに12パターンから**等確率でランダム選択**する（毎回再抽選）
- コマ幅合計は必ず 1.0
- コマ効果（`koma_effect_type`）は今回設定しない（`None` 固定）
- `koma1_effect_target_side` はエフェクトなしでも `All` を設定する

## 作成時に確認する項目

| 項目 | 内容 |
|------|------|
| 作品ID | 作品ID（例: kai, dan, spy） |
| 連番 | `normal_00001〜000XX`（ブロック数分） |
| 各ブロックの雑魚構成 | ブロックごとの雑魚キャラID・色属性・体数（ブロックごとに異なる場合は個別確認） |

---

## ガードレール

1. **行パターンはランダム選択**: `コマ設計_行パターン.csv` の12パターンの中からマスタデータ生成時に等確率でランダムに選ぶ（毎回再抽選）
2. **コマ効果は設定しない**: `koma_effect_type` は今回 `None` 固定。コマ効果の実装は対象外
3. **アウトポストHPは変更不可**: 100固定
4. **フェーズ切り替え禁止**: `SwitchSequenceGroup` は使用しない
5. **IDの一貫性**: MstInGame.id = MstAutoPlayerSequence.sequence_set_id = MstPage.id = MstEnemyOutpost.id
6. **MstEnemyStageParameterのFK確認**: MstAutoPlayerSequence.action_value（SummonEnemy時）に設定するIDが、同バッチ内のMstEnemyStageParameterに存在することを確認する
