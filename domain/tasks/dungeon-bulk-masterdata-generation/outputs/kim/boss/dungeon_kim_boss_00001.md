# dungeon_kim_boss_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

`dungeon_kim_boss_00001` は、「君のことが大大大大大好きな100人の彼女」（kim）シリーズの限界チャレンジ（dungeon）ボスブロック第1弾である。敵ゲートHPは **1,000**（dungeonボスブロック固定値）で、**ボスが撃破されるまでは敵ゲートへのダメージが無効**となる。BGMは `SSE_SBG_003_009`（メイン）、ボスBGMは `SSE_SBG_003_007`（ボス登場時に切り替わり）を使用し、ループ背景アセットは `kim_00001`（kim専用背景）を採用する。

コマフィールドは **1行構成**（dungeonボスブロック固定）で、コマ背景として `kim_00001` を使用する。コマ効果（SlipDamage・AttackPowerDown等）は設定しない。1行の内訳は1コマ（幅1.0）のシンプルな全幅構成を想定する。

登場する敵は **ボス1体＋護衛雑魚1種類**。ボスは `c_kim_00001_kim1_challenge_Boss_Red`（溢れる母性 花園 羽々里 / URキャラ / 赤属性・Defenseロール・HP 50,000・攻撃力 100・移動速度 40）で、dungeonのボスブロック仕様に基づき「HPは通常登場スペックより低め、攻撃力は高め」の調整を行うことが推奨される（実際の係数はスキルで決定）。花園 羽々里は `chara_kim_00001` に対応するURキャラであり、challenge_00004 や savage_00003 の最終ボスとしても登場する最高難易度の敵キャラクターである。護衛雑魚は `e_glo_00001_kim1_challenge_Normal_Colorless`（無属性・Attackロール・HP 10,000・攻撃力 100・移動速度 40）で、時間差で出現しボスを支援する役割を担う。

シーケンス構成はグループ切り替えなし（デフォルトグループ単一構成）で **3〜4行**程度を想定する。バトル開始直後（InitialSummon または ElapsedTime 0ms）にボスの花園 羽々里が敵ゲート前に配置され、ゲートに向かって進軍してくる（タワーオブフェンス形式の能動的構成）。以降 ElapsedTime で護衛ファントム（無属性）が時間差で1〜2体ずつ出現し、ボスへの集中攻撃を阻害する設計とする。ボスが撃破されるまでは `is_summon_unit_outpost_damage_invalidation = 1` 設定により敵ゲートのHPが削れない。ボス撃破後にゲートHPが削られ、最終的なクリア判定が行われる。

バトルヒントは「ボスが生き残っている間は敵ゲートにダメージが入らない！まずは溢れる母性 花園 羽々里を倒すことに集中しよう！」旨を案内する。ステージ説明文のイメージは「君のことを想い続ける彼女たちの頂点、花園 羽々里が立ちはだかる！ボスを撃破しなければ敵ゲートは壊せないぞ！青属性キャラは赤属性のボスに対して有利に戦えるぞ！」といった内容を想定する。作品世界観として、100人の彼女たちを束ねるリーダー的な存在の羽々里が最後の守護者として登場するシーンを演出する。
