# dungeon_kim_normal_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

`dungeon_kim_normal_00001` は、「君のことが大大大大大好きな100人の彼女」（kim）シリーズの限界チャレンジ（dungeon）通常ブロック第1弾である。敵ゲートHPは **100**（dungeon通常ブロック固定値）でダメージ有効。BGMは `SSE_SBG_003_002`、ループ背景アセットは `kim_00001`（kim専用背景）を使用する。

コマフィールドは **3行構成**（dungeon通常ブロック固定）で、各行にはコマ背景として `kim_00001` を使用する。コマ効果（SlipDamage・AttackPowerDown等）は設定しない。コマ行の幅割りは行1が2コマ構成（幅0.6＋幅0.4）、行2が1コマ（幅1.0）、行3が2コマ（幅0.5＋幅0.5）の3行を想定する。

登場する敵は **GLO汎用エネミー `enemy_glo_00001` ベースの2種類**（kim専用の雑魚エネミーは存在しないため汎用エネミーを採用）。メイン敵は `e_glo_00001_kim1_challenge_Normal_Colorless`（無属性・Attackロール・HP 10,000・攻撃力 100・移動速度 40）で、主力として複数体が時間差で出現する。サブ敵は `e_glo_00001_kim1_challenge_Normal_Red`（赤属性・Attackロール・HP 10,000・攻撃力 100・移動速度 40）で、中盤以降に混入してプレイヤーの属性対応を求める役割を担う。両者ともに攻撃型で前進圧力をかける動きをする。無属性は属性相性が均一なため序盤の慣らしに適しており、赤属性は青属性キャラで有利を取れることをプレイヤーに意識させる設計となっている。

シーケンス構成はグループ切り替えなし（デフォルトグループ単一構成）で **5〜6行**程度を想定する。バトル開始直後（ElapsedTime 300ms）から無属性汎用ファントムが1体出現し、以降 ElapsedTime により時間差で追加召喚される。中盤（2,000ms前後）から赤属性のファントムが混入し、終盤（4,000ms以降）は無属性・赤属性が交互に登場する波状構成とする。最終行では無属性ファントムを数体まとめて召喚し、プレイヤーへの最終的な圧力をかけて締めくくる。トリガーは全て `ElapsedTime` を使用し、グループ切り替えは行わない。

バトルヒントは「青属性キャラは赤属性の敵に対して有利に戦える」旨を案内する。ステージ説明文のイメージは「無属性と赤属性のファントムが時間差で次々と押し寄せてくるぞ！青属性キャラを編成して赤属性の敵に備えよう！」といった内容を想定する。作品世界観として、花園家の"彼女たち"が守護するような印象を持たせる設計とする。
