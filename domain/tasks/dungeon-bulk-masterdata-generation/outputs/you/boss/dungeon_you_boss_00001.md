# インゲーム要件テキスト — dungeon_you_boss_00001（ボスブロック）

幼稚園WARS（you）シリーズの限界チャレンジボスブロック第1弾（dungeon_boss）。敵ゲートHPは1,000（固定）で、ボスが撃破されるまで敵ゲートへのダメージは無効（ボス撃破型）。BGMは`SSE_SBG_003_001`（通常）、ボス登場時は`SSE_SBG_003_004`に切り替わる。ループ背景は`you_00001`。1行構成のコマフィールド（固定）で、コマ効果は設定しない。

メインボスはURキャラ「元殺し屋の新人教諭 リタ」（`chara_you_00001`）。強敵オーラ（unit_kind=Boss）付きで登場する。幼稚園WARSの元殺し屋という設定を活かし、凄まじい殺気と戦闘技術を持つ難敵として演出する。dungeon_boss仕様に基づき、通常登場より低いHP（10,000程度、`c_you_00001_you1_charaget01_Boss_Colorless`のbase HP基準）・高い攻撃力（500程度、savage版相当）で設定する。属性は赤（Red）とし、青属性キャラが有利に戦える構成とする。移動速度は速め（move_speed=45）。攻撃コンボは6回程度。必殺ワザ（スキル）あり。ボスが生存している間は敵ゲートへのダメージが無効となるため、プレイヤーはまずリタの撃破を最優先に動く必要がある。

ボスは`InitialSummon`でゲート前に配置し、能動的にプレイヤー側へ進軍してくる構成。護衛として`enemy_glo_00001`（GLO汎用敵 / 組織の刺客イメージ、無属性）が`ElapsedTime`トリガーで時間差出現し、ボスとの同時対処でプレイヤーへの圧力を高める。シーケンス行数は3〜4行（ボス1行＋護衛雑魚2〜3行）。限界チャレンジのタワーディフェンス形式（生存全員出撃可）で、プレイヤーは総力を挙げてリタを打ち倒す設計とする。「かつて最強の殺し屋だった先生」という幼稚園WARSの核心的な設定を体現する緊迫したボス戦を演出する。

バトルヒント・ステージ説明には「赤属性の強敵（ボス）が登場する（青属性有利）」「必殺ワザを使ってくる元殺し屋に注意、ボスを先に倒さないとゲートにダメージが入らない」旨を記載する。
