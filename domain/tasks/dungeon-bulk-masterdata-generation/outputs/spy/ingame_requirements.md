# SPY×FAMILY 限界チャレンジ インゲーム要件

### 通常ブロック（dungeon_normal）

- コンテンツ種別: dungeon（限界チャレンジ）
- シリーズ: spy
- 雑魚敵:
  - 主力: `enemy_spy_00001`（3〜4体）
  - サブ: `enemy_spy_00101`（2〜3体）
- HP: 1,000
- 攻撃力: 5,000
- 移動速度: 少し速い
- 必殺ワザ: なし
- コマ効果: なし
- 背景アセット: `spy_00005`
- BGM: `SSE_SBG_003_002`
