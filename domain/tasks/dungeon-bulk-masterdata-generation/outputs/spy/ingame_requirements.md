# SPY×FAMILY 限界チャレンジ インゲーム要件

### 通常ブロック（dungeon_normal）

| 項目 | 値 |
|------|----|
| コンテンツ種別 | `dungeon` |
| シリーズID | `spy` |
| 雑魚敵（主力） | `enemy_spy_00001`（3〜4体） |
| 雑魚敵（サブ） | `enemy_spy_00101`（2〜3体） |
| 雑魚 HP | 1,000 |
| 雑魚 攻撃力 | 5,000 |
| 雑魚 移動速度 | 少し速い |
| 雑魚 必殺ワザ | なし |
| コマ効果 | なし |
| 背景アセット | `spy_00005` |
| BGM | `SSE_SBG_003_002` |
