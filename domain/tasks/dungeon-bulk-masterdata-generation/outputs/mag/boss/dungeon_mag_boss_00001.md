# dungeon_mag_boss_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

`dungeon_mag_boss_00001` は、株式会社マジルミエ（mag）シリーズの限界チャレンジ・ボスブロック第1弾として設計されたインゲームバトル設定である。BGMには `SSE_SBG_003_001`（マジルミエ通常戦闘BGM）を使用し、ループ背景アセットキーは通常ブロックと同様に `mag_00004` を採用する。dungeon仕様に基づき、敵ゲート（MstEnemyOutpost）のHPは **1,000**（固定値）で設定されている。

コマフィールドは **1行固定**（dungeon ボスブロック仕様）で構成され、特殊コマ効果は設定しない。ボスブロックはタワーディフェンス（総力戦）形式で、生存している全キャラを投入して戦う既存インゲームルールと同様のゲーム性となる。敵ゲートはボスが撃破されるまでダメージ無効となっており、プレイヤーは必ずボスを倒してから敵ゲートに攻撃が通る仕組みである（`is_summon_unit_outpost_damage_invalidation=1`）。

ボスキャラクターには、mag作品のURキャラのうち **桜木 カナ**（`chara_mag_00001`）のdungeon向け challengeスペック版（`c_mag_00001_challange_Boss_Yellow`）を採用する。通常インゲームに登場するスペックより**HPは低く・攻撃力は高め**というdungeon仕様の設計方針に従い、HP=100,000・攻撃=500・移動速度=35・ロール=Boss・属性=黄（Yellow）で設定される。属性を黄属性とすることで、緑属性キャラが有利に戦える対応関係が生まれ、プレイヤーに編成の工夫を促す。桜木 カナは「新人魔法少女」というキャラクター設定を持つため、ボスとして立ちはだかりながらも成長途上の雰囲気を感じさせる演出が期待できる。

護衛雑魚として **冷却系怪異**（`enemy_mag_00001`）の無属性版（`e_mag_00001_general_Normal_Colorless`）を時間差で投入する。バトル開始と同時（InitialSummon）にボスの桜木 カナが敵ゲート前に配置され進軍を開始する。その後、3,000ms後に冷却系怪異（無属性）が1体、さらに6,000ms後にもう1体と計2体が時間差で追加される。護衛雑魚はボスを援護しつつプレイヤーの注意を引き、両者を同時に相手取る戦略判断を求める。全体で3〜4行のシーケンスにまとめる。グループ切り替えは行わない。

バトルヒントは未設定とする（実装時に要追加）。ステージ説明文には「黄属性のボスが登場するので緑属性のキャラは有利に戦える」「ボスを倒すまで敵ゲートへのダメージは無効」という2点の重要情報を明示し、ボスキャラクターがマジルミエの世界で活躍する魔法少女の一人であることを彷彿とさせる文面を添える。

### ステージ説明文イメージ

- **属性情報**: 黄属性のボスが登場するので緑属性のキャラは有利に戦える。無属性の護衛も登場する
- **ギミック情報**: ボスを倒すまで敵ゲートへのダメージは無効。まずボスを集中撃破せよ
- **世界観**: マジルミエの新人魔法少女・桜木カナが立ちはだかる決戦フロア

### 設計補足

- ボスのHPは100,000と控えめ（通常インゲームの `e_mag_00001_general2_Boss_Blue` HP=1,000,000と比較して大幅に低い）。dungeon仕様「通常より低いHP・高い攻撃力」の方針に合致する。
- 攻撃力=500はチャレンジスペックの標準値であり、dungeon難易度の進行係数が乗ることで実際の強さが決まる。
- 護衛雑魚に冷却系怪異（無属性）を選択することで、属性相性に関わらず均等に脅威となる設計。緑属性キャラでボスに挑みながら、無属性の護衛もさばく必要が生まれ戦略的な判断が求められる。
- InitialSummon でボスをゲート前に即配置することで、タワーディフェンス形式の「防衛戦」としての緊張感を開幕から演出する。
- `chara_mag_00201`（絶対効率の体現者 土刃 メイ / Red / `c_mag_00201_challange_Boss_Green`）は boss_00002 以降での登場を想定し、本ブロックでは桜木 カナを優先採用する。
