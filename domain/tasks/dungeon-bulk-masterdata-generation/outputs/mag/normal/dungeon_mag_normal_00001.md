# dungeon_mag_normal_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

`dungeon_mag_normal_00001` は、株式会社マジルミエ（mag）シリーズの限界チャレンジ・通常ブロック第1弾として設計されたインゲームバトル設定である。BGMには `SSE_SBG_003_001`（マジルミエ通常戦闘BGM）を使用し、ループ背景アセットには `mag_00004`（normalステージの序盤〜中盤で最も多用される作品固有背景）が設定されている。dungeon仕様に基づき、敵ゲート（MstEnemyOutpost）のHPは **100**（固定値）で、ダメージ無効化は適用されない（通常型）。

コマフィールドは **3行固定**（dungeon通常ブロック仕様）で構成され、いずれの行にも特殊コマ効果（SlipDamage・AttackPowerDown等）は設定されない。プレイヤー操作キャラが各フロアに進軍した際に敵が動き出す、受動的なアクションRPG形式で進行する。

登場する敵は2種類。メイン雑魚として **冷却系怪異**（`enemy_mag_00001`）の無属性版（`e_mag_00001_general_Normal_Colorless`）が出現し、HP=70,000・攻撃=1,200・移動速度=35の攻撃型（Attack）として機能する。中速で前線に迫るのが特徴で、耐久力と火力を兼ね備えたmag作品の標準雑魚敵である。サブ雑魚として **つらら**（`enemy_mag_00101`）の青属性版（`e_mag_00101_general_Normal_Blue`）が追加召喚される。つららはHP=10,000と撃破はしやすいが、移動速度=100と全作品中最速クラスの超高速雑魚であり、攻撃=1,500と火力も高い。少数でも侮れない脅威となる。

シーケンス構成はグループ切り替えなしの単一グループで、dungeon仕様に従い `ElapsedTime` による時間差配置を基本とする。バトル開始から500ms後に冷却系怪異（無属性）が1体出現し、プレイヤーに最初の対処を迫る。その後、2,000ms後・4,000ms後と段階的にさらに各1体ずつ出現し、最後に冷却系怪異が倒れた直後（または6,000ms後）のタイミングでつらら（青）が3体まとめて時間差（300ms間隔）で高速突撃してくる形で構成する。全体で5〜7行程度のコンパクトなシーケンスにまとめる。グループ切り替えは行わない。

バトルヒントはまだ整備中のため未設定とする（実装時に要追加）。ステージ説明文には「青属性の敵も登場するので黄属性のキャラは有利に戦える」「後半は超高速の怪異が突撃してくるので注意」という情報を盛り込み、魔法少女たちが怪異と戦うマジルミエの世界観を反映した文面とする。

### ステージ説明文イメージ

- **属性情報**: 青属性の敵も登場するので黄属性のキャラは有利に戦える
- **ギミック情報**: 後半に超高速の怪異（つらら）が突撃してくるので注意
- **世界観**: 魔法少女たちが怪異を退治するマジルミエの世界観を反映

### 設計補足

- メイン雑魚の冷却系怪異（無属性）は属性相性が発生しないため、どの属性でも均等に戦える序盤〜中盤の標準雑魚として適切。
- サブ雑魚のつらら（青属性・speed=100）は、黄属性のキャラで有利に戦える設計。速度が極めて高いため、プレイヤーに「素早く反応する」スキルを求める難度調整となっている。
- dungeon通常ブロックはアクションRPG（1体操作）形式のため、大量召喚（count=99）ではなく、少数精鋭の時間差配置で緊張感を演出する設計方針とした。
