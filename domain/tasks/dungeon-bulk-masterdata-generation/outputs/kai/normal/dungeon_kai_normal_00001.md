# dungeon_kai_normal_00001 — インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

怪獣８号（kai）シリーズの限界チャレンジ・通常ブロック第1弾（dungeon_normal）。敵ゲートHPは100（dungeon通常ブロック固定値）でダメージ有効。BGMは `SSE_SBG_003_001`（kai全コンテンツ共通の怪獣８号専用BGM）、ループ背景は `kai_00001`（kai作品専用背景）を使用する。3行固定のコマフィールドで構成され、コマアセットはすべて `kai_00001`。コマ効果（SlipDamageやAttackPowerDown等）は設定しない。

登場する敵は1種類。防衛省怪獣討伐部隊が立ち向かう怪獣・余獣（enemy_kai_00101）のColorless（無属性）バリアントで、パラメータIDは `e_kai_00101_general_Normal_Colorless`。HP 25,000・攻撃力 350・移動速度 45 の防衛型（Defense / Normal）であり、比較的標準的なスピードで砦に向かってくる中型の怪獣雑魚として機能する。dungeon通常ブロックはアクションRPGの1体操作バトルであるため、プレイヤーが1キャラクターを駆使して3体前後の余獣を順次撃破していく設計とする。

シーケンス構成はグループ切り替えなしの単一デフォルトグループ。ElapsedTimeトリガーのみで余獣を時間差配置し、シーケンス行数は4行程度を目安とする。バトル開始 500ms 後に余獣を1体召喚し、2,000ms 後・4,000ms 後にそれぞれ1体ずつ追加召喚することで計3体が段階的に前進する構成とする。これにより、1体ずつ丁寧に対処しながら徐々にプレッシャーが増す進行感を演出できる。砦ダメージが入ると追加召喚や強化ウェーブを入れる必要はなく、シンプルな3〜4シーケンスで完結させる設計を基本とする。

バトルヒントは「無属性の怪獣余獣が出現する。どの属性のキャラでも戦いやすいが、素早く撃破してHP消耗を抑えよう。」といったイメージ。ステージ説明文は「怪獣討伐部隊として余獣の群れに立ち向かえ！無属性の怪獣が次々と押し寄せてくる。」といった怪獣8号の世界観に沿った文言で統一する。
