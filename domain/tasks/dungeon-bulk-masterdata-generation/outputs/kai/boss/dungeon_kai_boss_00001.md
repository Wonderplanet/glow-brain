# dungeon_kai_boss_00001 — インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

怪獣８号（kai）シリーズの限界チャレンジ・ボスブロック第1弾（dungeon_boss）。敵ゲートHPは1,000（dungeonボスブロック固定値）でダメージ有効だが、ボスが撃破されるまで敵ゲートへのダメージは無効化される（is_summon_unit_outpost_damage_invalidation）。BGMは `SSE_SBG_003_001`（kai全コンテンツ共通）、ループ背景は `kai_00001`（kai作品専用背景）を使用する。コマフィールドは1行固定、コマアセットは `kai_00001`、コマ効果は設定しない。

登場するボスは `c_kai_00001_general_Normal_Yellow`、すなわち日比野 カフカ（黄属性 / Technical / Normal）。dungeon仕様に従い通常登場スペックより低HP・高攻撃力に設定することを念頭に置き、HP 150,000・攻撃力 700・移動速度 35 というパラメータIDそのままの値を採用する。カフカは防衛省の一般隊員から突然怪獣に変身した主人公であり、「変身してしまった自分との葛藤」を体現するボスとして登場する演出とする。ボスブロックはタワーディフェンス型の総力戦で、プレイヤーは手持ちの全キャラを投入して戦う構成となる。

護衛として余獣（enemy_kai_00101）のColorless（無属性）バリアント `e_kai_00101_general_Normal_Colorless`（HP 25,000・攻撃力 350・速度 45）が時間差で出現し、カフカを側面から援護する。これにより、プレイヤーはボス撃破を最優先としながら雑魚の流れ込みにも対処するマルチタスクが求められる。

シーケンス構成はグループ切り替えなしの単一デフォルトグループ。シーケンス行数は3行程度を目安とする。まずバトル開始直後（InitialSummon、条件値=0）にカフカ（ボス）を1体召喚し、その後 3,000ms 経過で余獣を1体、さらに 6,000ms 経過で余獣をもう1体召喚する構成とする。カフカが撃破されるまで敵ゲートにダメージが通らないため、プレイヤーはカフカを最優先で集中攻撃する必要がある。カフカ撃破後はゲートへのダメージが有効になり、そのままゲートを破壊してクリアとなる。

バトルヒントは「カフカが撃破されるまで敵ゲートはダメージを受けない。まず日比野カフカを倒すことに集中しよう！」といった内容。ステージ説明文は「防衛省の隊員・日比野カフカが立ちはだかる！黄属性のボスに緑属性のキャラが有利。護衛の怪獣余獣にも注意せよ。」といった怪獣8号の世界観に沿った文言とする。
