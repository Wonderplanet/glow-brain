# dungeon_gom_boss_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

`dungeon_gom_boss_00001` は、姫様"拷問"の時間です（gom）シリーズの限界チャレンジ・ボスブロック第1弾である。dungeon仕様の固定値として敵ゲートHPは1,000に設定されており、ボス（囚われの王女 姫様）が撃破されるまで敵ゲートへのダメージは無効化される仕様（`is_summon_unit_outpost_damage_invalidation=1`）となる。BGMには `SSE_SBG_003_006`（gom作品専用BGM）を使用し、ループ背景アセットには `gom_00001` を設定する。プレイヤーアウトポストアセットは `gom_ally_0001`。

コマフィールドは1行固定（dungeon ボスブロック仕様）で、コマアセットには `gom_00001` を使用する。コマ効果は設定なし（None）。ボスフロアは既存インゲームルールと同様のタワーオブフェンス形式（総力戦）であり、生存している全キャラが出撃可能となる。

ボスとして登場するのは **囚われの王女 姫様**（`c_gom_00001_general_n_Boss_Yellow`）。黄属性・Defenseロールで、HP基準値10,000・攻撃力50・移動速度25（低速）のゆったりとした前衛型。dungeon仕様では「通常登場よりHPは低め・攻撃力は高め」の設計方針に従い、HP係数は抑えめ（hp_coef=1程度）・攻撃係数はやや高め（attack_coef=2〜3程度）を想定する。姫様は重厚な存在感でゆっくりとゲートに向かって迫り、プレイヤーはボスを早期に撃破することでゲートへのダメージルートを開く必要がある。

護衛雑魚として **バタートースト**（`e_gom_00501_general_n_Normal_Colorless`）が時間差で出現し、姫様の前進をサポートする。ボスが登場した直後から護衛雑魚が断続的に召喚され、プレイヤーは姫様の処理と護衛雑魚の対応を同時にさばくことが求められる。護衛雑魚のHP係数・攻撃係数はdungeon通常ブロックに準じた抑えめの設定（hp_coef=1.5〜2程度）とし、あくまでもボスへの集中を妨げる役割に留める。

シーケンス構成はグループ切り替えなしのシングルグループで、全2〜3行を目安とする。まず開幕直後（ElapsedTime=0またはInitialSummon）にボス「囚われの王女 姫様」を1体召喚し、続いて500ms〜1,000ms経過を目安にバタートーストを2〜3体・インターバル300ms程度で時間差配置する構成とする。ボスが生存している限り敵ゲートへのダメージは入らないため、プレイヤーは真っ先にボス撃破を狙うことになる。

バトルヒントは未設定。ステージ説明文には、「囚われの王女 姫様が登場！ボスを倒さないと敵ゲートにダメージが入らないぞ。黄属性の敵には緑属性で属性有利に戦おう！」といった世界観とギミックを案内する文言を設定するイメージ。黄属性のボスが登場するため、緑属性キャラを編成することで属性有利が得られることを明記する。
