# dungeon_jig_boss_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

地獄楽（jig）シリーズの限界チャレンジ・ボスブロック第1弾（dungeon_boss難度）。敵ゲートHPはdungeon仕様の固定値1,000で、ボスが撃破されるまでゲートへのダメージは無効化される。BGMは`SSE_SBG_003_003`（地獄楽通常BGM）、ループ背景は`jig_00002`を使用する。コマフィールドはdungeonボスブロック仕様の1行固定構成で、コマ効果は設定しない。

ボスブロックはタワーオブフェンス形式（総力戦）で、生存している全キャラが召喚可能な能動的構成となる。ボスキャラは`c_jig_00001_mainquest_Boss_Red`（がらんの画眉丸 / URキャラ）で、赤属性・テクニカルタイプ（Technical）。dungeon用に調整したパラメータとして、HPは通常登場より低め（base HP=5,000にhp_coef=4程度で実質HP約20,000）、攻撃力は通常より高め（atk_coef=3程度）を想定する。移動速度は41と高速で、素早い進軍で前線を圧迫してくる。コンボ数は5回の多段攻撃が特徴で、地獄楽の伝説的剣士「がらんの画眉丸」の剣技が炸裂する迫力あるボス体験を提供する。

護衛雑魚として門神（`e_jig_00001_mainquest_Normal_Colorless`）が時間差で出現し、ボスの進軍を援護する。シーケンス構成はグループ切り替えなしのデフォルトグループ単一構成で、シーケンス行数は3行を想定。バトル開始直後にボス（がらんの画眉丸）をInitialSummonでゲート前に配置し、ElapsedTimeにより2,000ms後と5,000ms後に門神が護衛として時間差出現する。ボスが撃破されない限り敵ゲートへのダメージは無効のため、プレイヤーはボス撃破を最優先に動く必要がある。ボスブロックの設計意図として、dungeon通常より弱め（しかし十分な脅威感）のボスHPと高めの攻撃力によって、短期決戦的な緊張感あるバトルを演出する。

バトルヒントは未設定。ステージ説明では「がらんの画眉丸が高速で迫ってくる！ボスを倒すまでゲートにダメージが入らないので、まずはボスの撃破を優先しよう！青属性のキャラが赤属性の敵に有利だ！」というイメージで、ボスへの集中攻撃と属性相性（青属性有利）の活用を案内する。
