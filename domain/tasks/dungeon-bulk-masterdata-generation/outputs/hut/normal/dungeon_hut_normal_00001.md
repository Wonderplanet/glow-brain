# dungeon_hut_normal_00001 インゲーム要件テキスト

---

## パート1: 要件テキスト（日本語散文）

`dungeon_hut_normal_00001` は、限界チャレンジ（dungeon）コンテンツにおける『ふつうの軽音部』（hut）シリーズの通常ブロック（Normal Block）第1弾である。敵ゲートのHPは dungeon 仕様の固定値である **100** に設定されており、ダメージ有効（砦破壊型）としてプレイヤーは雑魚敵をさばきつつゲートを破壊してクリアを目指す。BGMには『ふつうの軽音部』の通常バトル曲 **`SSE_SBG_003_002`** を使用し、ループ背景アセットキーは未設定（hut シリーズは作品専用の背景ループアセットが存在しないため空白）とする。コマアセットには challenge・PVP でも使用実績のある汎用アセット **`glo_00014`** を採用する。

コマフィールドは dungeon 通常ブロックの仕様に従い **3行固定**。コマ効果は `None`（特殊効果なし）で統一する。『ふつうの軽音部』の軽音楽部活動という日常的な世界観に合わせ、シンプルで見通しの良い盤面構成とする。

登場する敵は1種類。GLO汎用敵 **`enemy_glo_00001`**（Colorless / Attack / Normal）を主力雑魚として使用する。パラメータIDは `e_glo_00001_hut1_challenge_Normal_Colorless` を参考に dungeon 向けに新規定義し、HP **1,000**・攻撃力 **100**・移動速度 **40**・待機距離 **0.2** とする。dungeon 仕様のゲートHP=100 に対して雑魚が到達するだけで即ゲームオーバーのリスクがあるため、enemy_hp_coef は **1.0** として素のHPで登場させる。役割は攻撃型（Attack）の前衛突進タイプであり、プレイヤーが操作する1体のキャラで次々と迎撃していく展開となる。

シーケンス構成は **グループ切り替えなし**（デフォルトグループのみ）、**ElapsedTime トリガー**による時間差配置を軸とした **5〜6行**の構成を想定する。バトル開始から 500ms 後に雑魚1体が先行出現し、プレイヤーがゲームに慣れる時間を確保する。その後 1,000ms・2,000ms・3,000ms・4,500ms と間隔を広げながら追加の雑魚が1体ずつ出現し、徐々に圧力を高めていく。dungeon 通常ブロックはアクションRPG（1体操作）のため、敵が連続して押し寄せすぎないよう間隔に余裕を持たせた設計とする。

バトルヒントは「無属性の敵が登場するぞ！素早く迎撃して限界チャレンジを突破せよ！」というイメージ。ステージ説明文は「無属性の敵が次々と現れる。キャラを上手に操作して敵を倒し、ゲートを守り抜こう！」というシンプルな内容とし、dungeon の1体操作という特性を強調する。
