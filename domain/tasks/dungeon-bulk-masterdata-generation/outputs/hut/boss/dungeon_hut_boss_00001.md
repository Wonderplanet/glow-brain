# dungeon_hut_boss_00001 インゲーム要件テキスト

---

## パート1: 要件テキスト（日本語散文）

`dungeon_hut_boss_00001` は、限界チャレンジ（dungeon）コンテンツにおける『ふつうの軽音部』（hut）シリーズのボスブロック（Boss Block）第1弾である。敵ゲートのHPは dungeon 仕様の固定値である **1,000** に設定されており、**ボスが撃破されるまで敵ゲートへのダメージは無効**（`is_summon_unit_outpost_damage_invalidation = 1`）となる。ボスを倒すことが最優先の課題であり、全キャラを出撃させるタワーディフェンス形式の総力戦として設計されている。BGMには通常バトル曲 **`SSE_SBG_003_002`** を使用し、ループ背景アセットキーは未設定（hut 専用背景ループアセット未存在のため空白）、コマアセットには **`glo_00014`** を採用する。

コマフィールドは dungeon ボスブロックの仕様に従い **1行固定**。コマ効果は `None`（特殊効果なし）とする。フロア1段のシンプルな構成で、プレイヤーが総力を結集してボスに挑む緊張感あるバトルを演出する。

ボスとして登場するのは hut シリーズのURキャラ **`chara_hut_00001`**（ひたむきギタリスト 鳩野 ちひろ）。属性は **Colorless（無属性）**、ロールは `Defense`（防衛型、URキャラの固定特性）、character_unit_kind は `Boss`。dungeon 仕様に従い「通常登場よりもHPは低め・攻撃力は高め」に設定し、HP **5,000**・攻撃力 **300**・移動速度 **35**・待機距離 **0.21**・攻撃コンボサイクル **5** とする。パラメータIDは dungeon 専用の新規定義（`c_hut_00001_dungeon_boss_Boss_Colorless`）を推奨する。`is_summon_unit_outpost_damage_invalidation = 1` を設定し、鳩野ちひろが場にいる間は敵ゲートが削れない構成とする。ギターを抱えた少女が力強く前線に立ちふさがり、プレイヤーの全キャラを相手取るイメージである。

護衛雑魚として **`enemy_glo_00001`**（Colorless / Attack / Normal）を時間差で出現させる。HP **1,000**・攻撃力 **100**・移動速度 **40**・待機距離 **0.2**。ボスとの戦いに割り込んでプレイヤーを圧迫する役割を担い、後方から続々と登場することでボス単体に集中させない設計とする。

シーケンス構成は **グループ切り替えなし**（デフォルトグループのみ）、**3行**の構成とする。バトル開始直後（0ms）にボス（鳩野ちひろ）を `InitialSummon` または `ElapsedTime=0` でゲート前に配置し、即座に進軍を開始させる。その後 2,000ms 後に護衛雑魚1体、さらに 4,000ms 後に護衛雑魚1体を追加召喚し、プレイヤーへの圧力を段階的に高めていく。ボスブロックは総力戦のため敵の出現間隔を短めに設定し、プレイヤーが全キャラを駆使して対処する緊張感を演出する。

バトルヒントは「ひたむきギタリスト 鳩野 ちひろが行く手を阻む！ボスを倒すまで敵ゲートにダメージが入らないぞ！全キャラで挑め！」というイメージ。ステージ説明文は「無属性の強敵が待ち構えている。ボスを撃破するまでゲートは守られる。全力で立ち向かおう！」とし、ボスのダメージ無効ギミックと総力戦のコンセプトをプレイヤーに伝える。
