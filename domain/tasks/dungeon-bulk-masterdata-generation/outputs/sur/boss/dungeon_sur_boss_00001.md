# dungeon_sur_boss_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

魔都精兵のスレイブ（sur）シリーズの限界チャレンジ・ボスブロック第1弾（dungeon_sur_boss_00001）。敵ゲートHPは1,000（dungeon固定値）でダメージ有効型。ボスが撃破されるまでは敵ゲートへのダメージは無効化される。BGMは`SSE_SBG_003_001`（surシリーズ通常バトル共通）、ループ背景アセットは`sur_00001`を使用する。

コマフィールドは1行固定のボスブロック仕様で構成され、コマアセットキーは`sur_00001`を使用する。コマ効果（特殊エフェクト）は設定しない。ボスブロックは既存インゲームルールと同様のタワーディフェンス方式で、生存している全キャラクターを召喚可能な総力戦形式となる。敵はゲート前に召喚されてから能動的に進軍してくる設計とする。

ボスには誇り高き魔都の剣姫・羽前 京香（`chara_sur_00101`）が登場する。dungeonボスブロックの仕様に基づき、通常登場スペックよりも低HP・高攻撃力で設定する。ボスパラメータIDは`c_sur_00101_general_Boss_Blue`を使用し、HP 400,000・攻撃力 700・移動速度 45・コンボ数 7回のAttackロール・Blue属性（青属性）として配置する。キャラクターとしての羽前 京香は魔都の精鋭戦士であり、高速で接近して連撃を浴びせる戦闘スタイルが特徴。dungeon仕様ではHPを抑えつつ攻撃力を高め、プレイヤーへの圧力が強い凶暴なボスとして機能させる。ボスには`is_summon_unit_outpost_damage_invalidation=1`を設定し、羽前 京香が生存している間は敵ゲートへのダメージが発生しない設計とする。

護衛雑魚として醜鬼（`enemy_sur_00101`）の無属性版（パラメータID: `e_sur_00101_general_Normal_Colorless`）が時間差で出現し、ボスを援護する。醜鬼はHP 3,000・攻撃力 100・移動速度 35のDefense型として、プレイヤーの注意を分散させる役割を担う。

シーケンス構成はグループ切り替えなしのデフォルトグループのみで完結し、全シーケンス行数は3行を想定する。バトル開始直後（0ms）にボスの羽前 京香が`InitialSummon`によってゲート前に1体配置される。ボスが撃破されるまで敵ゲートはダメージ無効状態が続くため、プレイヤーはまず羽前 京香の撃破を最優先として戦う必要がある。バトル開始から2,000ms後に護衛醜鬼（無属性）が2体、インターバル500msで時間差登場する。さらに4,500ms後には醜鬼が追加で2体出現し、ボスと雑魚が複合的な圧力をかけ続ける構成とする。MstInGameの係数はHP・攻撃・速度ともに1.0（等倍）で運用する。

バトルヒントは「誇り高き魔都の剣姫・羽前 京香がボスとして登場。ボスを撃破するまで敵ゲートにダメージが入らないぞ！まずは羽前 京香を倒せ！」という内容を想定する。ステージ説明文では「魔都最強の剣姫・羽前 京香が立ちはだかる。彼女を打ち倒し、限界チャレンジを突破せよ！」という作品世界観に即した文言を採用する。
