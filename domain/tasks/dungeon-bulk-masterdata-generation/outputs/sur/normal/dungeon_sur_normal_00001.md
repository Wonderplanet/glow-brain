# dungeon_sur_normal_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

魔都精兵のスレイブ（sur）シリーズの限界チャレンジ・通常ブロック第1弾（dungeon_sur_normal_00001）。敵ゲートHPは100（dungeon固定値）でダメージ有効型。BGMは`SSE_SBG_003_001`（surシリーズ通常バトル共通）、ループ背景アセットは`sur_00001`を使用する。

コマフィールドは3行固定のdungeon仕様で構成され、コマアセットキーは`sur_00001`を使用する。行数は3行で固定され、コマ効果（特殊エフェクト）は一切設定しない。プレイヤーは1体のみを操作する通常ブロック形式のバトルであり、味方キャラが各フロアに進行したタイミングで敵が動き出す受動的な構成となる。

登場する敵は醜鬼（`enemy_sur_00101`）のみで構成される。序盤は弱体の無属性（Colorless）醜鬼（パラメータID: `e_sur_00101_general_Normal_Colorless`）が登場し、HP 3,000・攻撃力 100・移動速度 35のDefense型として前線を固める。中盤以降は青属性（Blue）醜鬼（パラメータID: `e_sur_00101_general_Normal_Blue`）が加わり、HP 15,000・攻撃力 200・移動速度 40のAttack型として戦局を押し上げる。終盤には緑属性（Green）醜鬼（パラメータID: `e_sur_00101_general_Normal_Green`）が出現し、HP 22,000・攻撃力 300・移動速度 50のAttack型として最も強力な圧力をかけてくる。醜鬼は魔都を蝕む怪異の化身であり、3色の醜鬼が波状に押し寄せる構成によって作品世界の緊迫感を演出する。

シーケンス構成はグループ切り替えなしのデフォルトグループのみで完結し、全シーケンス行数は5行程度を想定する。すべてのトリガーは`ElapsedTime`（経過時間）による時間差配置を基本とし、開始直後から3段階で醜鬼の属性が強化される展開を設計する。バトル開始300ms後に無属性醜鬼3体が時間差で出現し、2,000ms経過後に青属性醜鬼2体が追加される。さらに4,500ms経過後には緑属性醜鬼1体が姿を現し、フロアの締めくくりとして赤属性キャラが有利に対応できる局面を作り出す（緑属性の敵には赤属性有利）。MstInGameの係数はHP・攻撃・速度ともに1.0（等倍）で運用する。

バトルヒントは「無属性、青属性、緑属性の醜鬼が次々と出現する。緑属性の醜鬼には赤属性のキャラが有利だぞ！」程度のシンプルな内容を想定する。ステージ説明文では「魔都から現れた醜鬼の群れが迫りくる。段階的に強化される敵の波を食い止めろ！」という世界観に沿った文言を採用する。
