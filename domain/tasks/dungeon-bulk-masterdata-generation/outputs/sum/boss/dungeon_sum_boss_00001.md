# dungeon_sum_boss_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

`dungeon_sum_boss_00001` は、サマータイムレンダ（sum）シリーズの限界チャレンジボスブロック第1弾である。コンテンツ種別は `dungeon（限界チャレンジ）`、ブロック種別は `boss`。敵ゲートHPは **1,000**（dungeonボスブロック固定値）で、ボスが撃破されるまで敵ゲートへのダメージは無効化される（`is_summon_unit_outpost_damage_invalidation = 1`）。BGMには `SSE_SBG_003_003`（sumシリーズ全コンテンツ共通）を使用し、ループ背景アセットには `sum_00003`（normalシリーズ終盤および最終ステージ `normal_sum_00006` と同じ背景）を設定する。ボスBGMは未設定であり、ボス登場時も通常BGMが継続する。

コマフィールドは **1行**（dungeonボスブロック固定値）で構成され、コマ効果は設定しない。限界チャレンジボスブロックはタワーディフェンス方式（総力戦）であり、生存している全キャラが召喚可能な既存インゲームルールに準じた構成となる。ボスは敵ゲート前に配置され、進軍してくる能動的な構成をとる。

ボスには `chara_sum_00101`（影のウシオ 小舟 潮）のURキャラをベースとした `c_sum_00101_general_Boss_Red`（赤属性）を使用する。HP 190,000・攻撃力 600・移動速度 40・コンボ数 7 の Support ロールで、dungeon仕様に準じてHPは通常登場スペックより低く、攻撃力は高く設定されている（既存パラメータIDをそのまま活用し、coef調整でボスブロックにふさわしい強度を実現する）。「影のウシオ」という存在感を持つURキャラが正面から立ちはだかる、緊張感のある対峙シーンを演出する。

護衛雑魚として `e_sum_00001_general_Normal_Colorless`（影 / 無属性）を時間差で出現させる。ボスが正面で圧力をかけながら、無属性の影たちが次々と現れてプレイヤーを挟み込む構図とする。ボスのコンボ数が7と多く、攻撃力が高いため、防衛に徹しながらボスを先に倒すことが攻略の鍵となる設計である。

シーケンス構成はグループ切り替えなしのデフォルトグループ単一構成で、行数は **3行**（目安2〜4行の範囲内）とする。バトル開始直後の `InitialSummon` でボス（`c_sum_00101_general_Boss_Red`）を1体ゲート前に配置し、1,500ms 後に `ElapsedTime` トリガーで護衛の影（`e_sum_00001_general_Normal_Colorless`）を2体時間差で出現させる。さらに 3,500ms 後にも護衛の影を1体追加召喚し、ボスが倒れるまで継続的なプレッシャーを与える全3行構成とする。

バトルヒントは「影のウシオを倒さないと敵ゲートにダメージが入らないぞ！青属性のキャラが赤属性の敵に有利だ」というイメージで設定する。ステージ説明文のイメージは「サマータイムレンダの世界から影のウシオが現れた！強力なボスを倒して先へ進め！赤属性の敵が登場するので青属性のキャラが有利に戦えるぞ」とする。
