# dungeon_osh_normal_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

【推しの子】（osh）シリーズの限界チャレンジ（dungeon）通常ブロック第1弾。インゲームIDは `dungeon_osh_normal_00001`。敵ゲートHPは dungeon 仕様の固定値 100 で、ダメージ有効（砦破壊型）。BGMは `SSE_SBG_003_003`（【推しの子】メインBGM）、ループ背景アセットは `osh_00001`（全normal共通）を使用する。

フィールドは dungeon 通常ブロック仕様の3行固定構成。行1は2コマ（幅0.5＋幅0.5）、行2は3コマ（幅0.33＋幅0.34＋幅0.33）、行3は2コマ（幅0.5＋幅0.5）で構成する。いずれのコマにも特殊効果（コマ効果）は設定しない。

登場する敵は2種類。いずれも汎用敵を使用（osh には専用雑魚敵が存在しないため）。

メイン雑魚は `e_glo_00001_general_osh_n_Normal_Yellow`（汎用敵1 / Yellow属性 / Attack / HP=1,000・攻撃100・速度34）で、中速で進軍する主力雑魚として機能する。サブ雑魚は `e_glo_00002_general_osh_n_Normal_Colorless`（汎用敵2 / 無属性 / Attack / HP=1,000・攻撃100・速度47）で、Yellow よりも高速（速度47）に接近してくる前衛プレッシャー役として機能する。

シーケンス構成はグループ切り替えなし・デフォルトグループのみの単一構成（dungeon 仕様）。シーケンス行数は5行を目安とする。`ElapsedTime` トリガーによる時間差配置を基本とし、バトル開始直後から Colorless 高速雑魚（`enemy_glo_00002`）が1体登場してプレイヤーに即時対応を迫る。その後、時間経過に応じて Yellow 雑魚（`enemy_glo_00001`）が複数体ずつ段階的に追加召喚され、フィールドへの圧力が徐々に増していく設計とする。オシャレなエンタメの世界観を体現するように、芸能界の華やかさを連想させるテンポよいウェーブ配置で統一する。

バトルヒントは未設定。ステージ説明文は「黄属性の敵と無属性の敵が登場するぞ！緑属性のキャラは黄属性の敵に対して有利に戦うことができるぞ！」といったイメージで、黄属性への属性相性の活用を示唆する内容とする。
