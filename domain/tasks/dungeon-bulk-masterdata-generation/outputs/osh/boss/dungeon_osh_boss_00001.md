# dungeon_osh_boss_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

【推しの子】（osh）シリーズの限界チャレンジ（dungeon）ボスブロック第1弾。インゲームIDは `dungeon_osh_boss_00001`。敵ゲートHPは dungeon 仕様の固定値 1,000 で、ボスが撃破されるまで敵ゲートへのダメージは無効（`is_summon_unit_outpost_damage_invalidation=1`）。BGMは `SSE_SBG_003_003`（【推しの子】メインBGM）、ループ背景アセットは `osh_00001` を使用する。

フィールドは dungeon ボスブロック仕様の1行固定構成。行1は全幅1コマ（幅1.0）で構成し、コマ効果は設定しない。タワーオブフェンス（総力戦）形式のため、生存している全キャラを召喚可能。

ボスキャラクターは `c_osh_00001_general_osh_n_Boss_Colorless`（B小町不動のセンター アイ / 無属性 / Technical / HP=1,000・攻撃100・速度20）を使用する。dungeon ボスブロック仕様に従い、通常登場よりも HP を低め・攻撃力は高めに設定するため、シーケンス内 HP 係数は 50（実質 HP=50,000 相当）、攻撃力係数は 20（実質攻撃力=2,000 相当）を想定する。アイはゆっくりとした速度（speed=20）で前進し、近接した味方キャラに対してコンボ攻撃を繰り出す。その圧倒的なカリスマ性と存在感がフィールドを支配するイメージで設計する。

護衛雑魚は `e_glo_00001_general_osh_n_Normal_Yellow`（汎用敵1 / Yellow属性 / Attack / HP=1,000・攻撃100・速度34）を使用。バトル開始から時間差で2〜3体を登場させ、ボスへの集中攻撃を妨害する役割を担う。

シーケンス構成はグループ切り替えなし・デフォルトグループのみの単一構成（dungeon 仕様）。シーケンス行数は3行を目安とする。バトル開始直後（InitialSummon または ElapsedTime 0ms 相当）にボス（アイ）が敵ゲート前に1体配置され、その後 `ElapsedTime` トリガーで護衛雑魚（Yellow汎用敵）が時間差で出現する設計とする。ボスが存在する間は敵ゲートにダメージが入らないため、プレイヤーはまずボスの撃破を最優先に立ち回る必要がある。星のような輝きを放つ伝説のアイドル「アイ」が、その歌声と演技で敵を圧倒するかのような演出をイメージした設計。

バトルヒントは未設定。ステージ説明文は「強力なボスを倒すまでは、敵ゲートにダメージが入らないぞ！まずはボスを撃破することを優先しよう！無属性の敵が登場するぞ！」といったイメージで、ボスを最優先撃破するプレイ方針を示唆する内容とする。
