# ダンダダン（dan）限界チャレンジ インゲーム要件テキスト

---

## dungeon_dan_normal_00001

ダンダダン（dan）シリーズの限界チャレンジ通常ブロック。砦HPは100でダメージ有効（砦破壊型）。BGMは`SSE_SBG_003_001`（ダンダダン作品BGM）、ループ背景は`dan_00007`。3行構成のコマフィールドを使用し、行1は2コマ（幅0.6＋幅0.4）、行2は3コマ（幅0.25＋幅0.5＋幅0.25）、行3は2コマ（幅0.6＋幅0.4）で、いずれのコマにも特殊効果は設定されていない。

登場する敵は2種類。前半に出現する変身型セルポ星人（enemy_dan_00001 の `_trans_` バリアント、Colorless属性）はHP1,000・攻撃50・速度34の攻撃型（Attack）で、一見するとか弱い1体に見える。後半に大量出現する変身後セルポ星人（enemy_dan_00101, Colorless）はHP10,000・攻撃50・速度47の攻撃型で、前半の個体より13ポイント速く、素早い突進が特徴。

グループ構成はデフォルトグループと group1 の2段階。デフォルトグループでは開幕250ms後に変身型セルポ星人が1体召喚され、プレイヤーによる撃破で変身イベント（FriendUnitTransform=1）が発生した瞬間に group1 へ切り替わる。group1 では変身後セルポ星人が切り替え直後から100ms・50ms・2,000ms の時間差で5体ずつ3波、計15体が一気に押し寄せる波状攻撃設計となっている。

バトルヒントは未設定。ステージ説明では「ダメージを受けると変身する敵が登場する」こと、変身をトリガーに新たな大群が押し寄せることが案内される。前半の変身型1体を慎重に処理するか、あえて即座に倒して後半の大波を早めに迎え撃つかの駆け引きがこのステージの核心。無属性統一のためどの属性のキャラでも参戦可能だが、大量出現する後半に備えて範囲攻撃の必殺ワザ持ちキャラの編成が有効。

## dungeon_dan_boss_00001

ダンダダン（dan）シリーズの限界チャレンジボスブロック。砦HPは1,000でダメージ有効（砦破壊型）。BGMは`SSE_SBG_003_001`、ループ背景は`dan_00007`。1行構成のコマフィールドで、行1は全幅1コマ（幅1.0）。コマ効果は設定されていない。

登場するボスはURキャラ「ターボババアの霊力 オカルン」（chara_dan_00002）で、Red属性・Attackロール。HP50,000・攻撃力高め・速度50のボス格で、強敵オーラ（unit_kind=Boss）持ち。「ターボ」の名を冠するキャラクターらしく移動速度は高水準に設定されており、砦へ素早く迫る突進型のボス。護衛として変身後セルポ星人（enemy_dan_00101, Colorless, HP10,000・速度47）が開幕500ms後に2体出現し、1体撃破後にさらに3体追加で押し寄せる。

グループ切り替えはなく、デフォルトグループのみのシンプルな構成。ボスが高速で迫る中、護衛の変身後セルポ星人が前衛として絶え間なく補充される設計。ボスのHPが高いため、護衛に構いすぎるとボスが砦に到達してしまう緊張感を生んでいる。

バトルヒントは未設定。ステージ説明では「赤属性のボスに対して青属性キャラが有利」であることが案内される。護衛の無属性（Colorless）セルポ星人には属性相性が発生しないため、ボスへの集中攻撃を優先しながら護衛を手早く処理する攻撃的な立ち回りが攻略の鍵となる。
