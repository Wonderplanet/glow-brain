# dungeon_dan_boss_00001

## パート1: 要件テキスト（日本語散文）

`dungeon_dan_boss_00001` は、ダンダダン（dan）シリーズの限界チャレンジ・ボスブロック第1弾として設計されたインゲームステージである。BGMには `SSE_SBG_003_001`（ダンダダン通常BGM）が使用され、ループ背景アセットには `dan_00005` が設定されている。敵ゲートのHPは **1,000**（dungeonボスブロック固定値）であり、ボスが生存している間はゲートへのダメージが無効化される（`is_summon_unit_outpost_damage_invalidation = 1`）。プレイヤーは必ずボスを撃破してから砦を崩すという二段構えの戦略が求められる。ボスBGMは設定せず、通常BGMが全編を通じて流れ続ける構成とする。

コマフィールドは **1行固定**（dungeonボスブロック仕様）で構成され、コマ効果（暗闇・ダメージ等の特殊効果）は設定しない。タワーディフェンス型（総力戦ルール）のボスブロックは全キャラ出撃可能であり、1行の幅広フィールドで敵と正面から激突する設計となっている。コマのレイアウトは全幅1コマ（幅1.0）のシンプル構成を採用する。

登場する敵は2種類で、ボスとして **ターボババアの霊力 オカルン**（`chara_dan_00002`）が出現する。dungeon仕様では通常登場時よりHPを低め・攻撃力を高めに設定する方針のため、バリアント `c_dan_00002_general_n_Boss_Blue`（Blue / Attack / HP=10,000、攻撃力=50ベース）をベースとしてHP係数を抑えた状態で配置する。ターボババアの霊力を宿したオカルンは青属性・攻撃型のURキャラであり、ダンダダン世界屈指の強敵としてボスフロアに君臨する。その圧倒的な存在感と高い攻撃力でプレイヤーを脅かす設計とする。

護衛雑魚として **セルポ星人（変身）**（`enemy_dan_00101`）が `ElapsedTime` トリガーで時間差登場する。バリアント `e_dan_00101_general_n_Normal_Colorless`（Colorless / Attack / HP=10,000、攻撃力=50、移動速度=47）を採用し、ボスへの集中を妨げる高速の援軍として機能させる。属性を無属性（Colorless）とすることで、属性カウンターに頼れない緊張感を演出する。

シーケンス構成はグループ切り替えなしの **2〜3行** 構成とする。バトル開始直後（0ms）にボスのターボババアの霊力 オカルンを `InitialSummon` でゲート前に1体配置する。続いて2,000ms後に護衛のセルポ星人（変身）を1体、さらに4,000ms後に同じくセルポ星人（変身）を1体追加召喚する。ボスが撃破されるまで敵ゲートはダメージ無効のため、プレイヤーはボスの撃破を最優先としながら後続の高速雑魚にも対応する必要がある。ダンダダンらしいオカルンの人外的な強さと、その周囲を取り巻くセルポ星人の援護という世界観が体現された戦闘構成となっている。

バトルヒントは未設定とする。ステージ説明文（MstInGameI18n）では「ボスを倒すまでゲートにダメージが通らないぞ！青属性の敵には黄属性キャラが有利。強力なボスを優先して撃破せよ！」というメッセージを設定し、敵ゲートダメージ無効ギミックへの注意と属性有利情報を提示する。
