# dungeon_yuw_normal_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

`dungeon_yuw_normal_00001` は、2.5次元の誘惑（yuw）シリーズの限界チャレンジ・通常ブロック第1弾として設計されたインゲームバトル設定である。インゲームIDは `dungeon_yuw_normal_00001`、ブロック種別は dungeon_normal（通常ブロック）。敵ゲートのHPは **100**（dungeon normal 固定値）、敵ゲートへのダメージは有効である。BGMには `SSE_SBG_003_006`（yuwシリーズ固有の通常バトルBGM）を使用し、ループ背景アセットには `yuw_00001`（yuwシリーズ固有背景）を設定する。ボスBGMは設定しない。

コマフィールドは **3行固定**（dungeon normal 仕様）で構成する。コマアセットには `yuw_00001` を使用し、コマ効果（Gust / AttackPowerDown / SlipDamage 等）は **設定しない**（None）。コマレイアウトの行構成は pvp_yuw の実績に準じ、行1を2コマ（幅 0.6＋幅 0.4）、行2を2コマ（幅 0.75＋幅 0.25）、行3を1コマ（幅 1.0）の計5コマで設計する。

登場する敵は2種類。主力は天乃 リリサ（`chara_yuw_00001`）の dungeon 専用パラメータ版で、Yellow属性・Attack ロール。HP は dungeon 向けに低め（既存 charaget が base_hp 50,000 のため、dungeon 用に新規作成: base_hp **10,000**、hp_coef=1.0 で実HP **10,000**）、攻撃力は **320**、移動速度は **34**（yuwキャラ中最速）に設定する。リリサはコスプレ世界に飛び込む真っすぐなヒロインとして、素早く前進する攻撃型の主力雑魚として機能する。

サブとして橘 美花莉（`chara_yuw_00101`）の dungeon 専用パラメータ版、Green属性・Technical ロールを用いる。base_hp **12,000**（実HP 12,000）、攻撃力 **280**、移動速度 **29** に設定する。美花莉はリリサより一歩後ろを歩む形で時間差出現し、技巧的な動きでプレイヤーを揺さぶるサブ役として機能する。

シーケンス構成はグループ切り替えなしのデフォルトグループ単一構成とし、全 **5行** で完結させる。バトル開始から 500ms 後に主力のリリサを1体召喚（ElapsedTime トリガー）し、2,000ms 後に2体目のリリサを追加召喚、4,000ms 後にサブの美花莉を1体投入、6,000ms 後にリリサをさらに1体追加し、8,000ms 後に美花莉でしめる構成とする。こうして Yellow → Yellow → Green → Yellow → Green と交互に属性が入れ替わりながら進軍する波状構成により、単一属性に偏った編成では対応が難しくなる設計を意図する。グループ切り替えおよびギミックオブジェクトは使用しない。

バトルヒントは「黄属性の敵には緑属性、緑属性の敵には赤属性が有利です」と設定する。ステージ説明文は「コスプレの世界へようこそ！天乃リリサたちが進軍してくるぞ。黄属性と緑属性の敵が入り混じって登場するため、両属性に対応できる編成で挑もう！」とする。
