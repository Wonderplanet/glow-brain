# dungeon_yuw_boss_00001 インゲーム要件テキスト

## パート1: 要件テキスト（日本語散文）

`dungeon_yuw_boss_00001` は、2.5次元の誘惑（yuw）シリーズの限界チャレンジ・ボスブロック第1弾として設計されたインゲームバトル設定である。インゲームIDは `dungeon_yuw_boss_00001`、ブロック種別は dungeon_boss（ボスブロック）。敵ゲートのHPは **1,000**（dungeon boss 固定値）。ボスが生存している間は**敵ゲートへのダメージが無効**（`is_summon_unit_outpost_damage_invalidation=1`）となるため、プレイヤーはまず最初にボスを撃破することが必須となる。BGMには `SSE_SBG_003_006`（yuwシリーズ固有の通常バトルBGM）を使用し、ボス出現と同時に `SSE_SBG_003_007`（yuwシリーズのボス専用BGM）へ切り替わる。ループ背景アセットには `yuw_00001` を設定する。

コマフィールドは **1行固定**（dungeon boss 仕様）で構成する。コマアセットには `yuw_00001` を使用し、コマ効果は **設定しない**（None）。行1を1コマ（幅 1.0）の全幅単一レイアウトとし、ボスフロアとして広いフィールドでの真正面決戦を演出する。

ボスとして登場するのは、勇気を纏うコスプレ 乃愛（`chara_yuw_00301`）の dungeon ボス専用パラメータ版。Blue属性・Technical ロール。dungeon 仕様「ボスのHPは通常より低め、攻撃力は高め」に従い、base_hp は **15,000**（実HP 15,000）と控えめに抑える一方、攻撃力を **800** と高めに設定する。移動速度は **29**、well_distance は **0.23** に設定する。乃愛はコスプレに情熱を傾ける2.5次元の申し子として、テクニカルな動きでプレイヤーに挑みかかるボスとして機能する。ボスには `is_summon_unit_outpost_damage_invalidation=1` を付与し、ボス撃破までゲートダメージ無効を実装する。

護衛雑魚には天乃 リリサ（`chara_yuw_00001`）の dungeon 専用パラメータ版（Yellow属性・Attack ロール）を用いる。base_hp **10,000**、攻撃力 **320**、移動速度 **34**。ボスを守るように時間差で出現し、プレイヤーの注意を分散させる役割を担う。

シーケンス構成はグループ切り替えなしのデフォルトグループ単一構成とし、全 **3行** で完結させる。バトル開始直後（0ms）にボスの乃愛を InitialSummon でゲート前に配置する（行1）。続いて 3,000ms 後にリリサを1体 ElapsedTime で護衛として召喚し（行2）、さらに 6,000ms 後にリリサをもう1体追加召喚する（行3）。こうしてボスを中心に護衛雑魚が時間差で押し寄せる設計とする。プレイヤーはボス（Blue属性）撃破を最優先にしつつ、押し寄せる Yellow属性の護衛リリサにも対応が求められる。ボスが撃破されるまで敵ゲートはダメージ無効のため、ボスへの集中攻撃が攻略の核心となる。グループ切り替えは行わない。

バトルヒントは「ボスを倒すまで敵ゲートにダメージが入りません！まずはボス（青属性）を撃破しよう。緑属性キャラが有利です」と設定する。ステージ説明文は「乃愛がボスとして立ちはだかる！ボスが生存している間は敵ゲートへのダメージが無効。ボスを撃破してから敵ゲートを破壊せよ。護衛の天乃リリサにも要注意！」とする。
