# インゲーム要件テキスト テンプレート

インゲームマスタデータ（MstInGame 等）を作成する際に、スキル（`/masterdata-ingame-creator`）に渡す要件テキストのテンプレート。

---

## どの情報が必要か（一覧）

| カテゴリ | 項目 | 必須/任意 | 備考 |
|---------|------|---------|------|
| **基本** | コンテンツ種別（ステージ種別） | 必須 | |
| **基本** | シリーズID | 必須 | 例: spy, kai, dan |
| **基本** | インゲームID または命名の根拠 | 任意 | 不明なら自動提案される |
| **雑魚敵** | キャラID（1体以上） | 必須 | enemy_xxx / chara_xxx |
| **雑魚敵** | HP | 必須 | 最終HPの目安値 |
| **雑魚敵** | 攻撃力 | 必須 | 最終攻撃力の目安値 |
| **雑魚敵** | 移動速度 | 必須 | 遅い/普通/少し速い/速い |
| **雑魚敵** | 必殺ワザ（アビリティ）の有無 | 必須 | |
| **雑魚敵** | 色属性 | 任意 | 無指定=Colorless |
| **ボス** | ボスキャラID・名称 | ボスありなら必須 | |
| **ボス** | HP | ボスありなら必須 | |
| **ボス** | 攻撃力 | ボスありなら必須 | |
| **ボス** | 移動速度 | ボスありなら必須 | |
| **ボス** | 必殺ワザ（アビリティ）の有無 | ボスありなら必須 | |
| **ボス** | 強敵オーラの有無 | ボスありなら必須 | |
| **ボス** | 色属性 | 任意 | 無指定=Colorless |
| **ボス** | 進軍トリガー | 任意 | 例: ダメージ1発受けたら進軍 |
| **ボス** | 変身の有無・変身HP% | 任意 | |
| **ボス** | 護衛雑魚の有無・出現タイミング | 任意 | |
| **コマ** | コマ効果の有無・種別 | 任意 | 無指定=なし(None) |
| **アセット** | 背景アセットキー | 任意 | 既存ステージから引用可 |
| **アセット** | BGMアセットキー | 任意 | 既存ステージから引用可 |
| **特別ルール** | SpeedAttack制限時間 | event_challenge等で必須 | |
| **特別ルール** | コンティニュー禁止 | 任意 | |
| **特別ルール** | シリーズ制限 | 任意 | |
| **その他** | release_key | 任意 | 不明なら999999999で暫定 |

## テンプレート A：dungeon（限界チャレンジ）用

```
### 通常ブロック（dungeon_normal）

- コンテンツ種別: dungeon（限界チャレンジ）
- シリーズ: {spy / kai / dan ...}
- 雑魚敵:
  - 主力: {enemy_xxx_00001}（{N}体）
  - サブ: {enemy_xxx_00101}（{N}体）※不要なら削除
- HP: {数値}
- 攻撃力: {数値}
- 移動速度: {遅い / 普通 / 少し速い / 速い}
- 必殺ワザ: {なし / あり}
- コマ効果: {なし / AttackPowerDown / Poison / Darkness ...}
- 背景アセット: {spy_00005 など}
- BGM: {SSE_SBG_003_002 など}

---

### ボスブロック（dungeon_boss）

- コンテンツ種別: dungeon（限界チャレンジ）
- シリーズ: {spy / kai / dan ...}
- ボスキャラ: {名前}（{chara_xxx_00101}）
- 強敵オーラ: {あり / なし}
- 色属性: {Colorless / Red / Blue / Yellow / Green}
- HP: {数値}
- 攻撃力: {数値}
- 移動速度: {遅い / 普通 / 少し速い / 速い}
- 必殺ワザ: {あり / なし}
- 変身: {なし / あり（HP{N}%以下で変身）}
- 護衛雑魚: {なし / あり（{N}秒後に {enemy_xxx} × {N}体など）}
- コマ効果: {なし}
- 背景アセット: {spy_00005 など}
- BGM: {SSE_SBG_003_002 など}
```

> **dungeon 固定値（記述不要）：**
> - dungeon_normal: アウトポストHP=100、コマ3行固定
> - dungeon_boss: アウトポストHP=1,000、コマ1行固定

## 速度感の目安

| 表現 | move_speed（参考値） |
|------|-------------------|
| 遅い | 20〜35 |
| 普通 | 35〜50 |
| 少し速い | 45〜55 |
| 速い | 50〜80 |

## HP・攻撃力の目安

| 種別 | HP目安 | 攻撃力目安 |
|------|--------|---------|
| 雑魚（通常） | 1,000〜10,000 | 200〜500 |
| 雑魚（高難易度） | 50,000〜300,000 | 500〜2,000 |
| ボス（標準） | 100,000〜500,000 | 300〜3,800 |
| dungeon雑魚 | 1,000〜5,000 | 5,000（dungeon専用高設定） |
| dungeonボス | 50,000〜100,000 | 10,000〜20,000 |

## 記述のポイント

- **キャラIDが不明な場合**は「SPY×FAMILYのロイド」のように名前で書いてOK。スキルが調査してIDを特定する。
- **アセット情報が不明な場合**は省略してOK。スキルが既存の同シリーズデータから引用する。
- **数値が曖昧な場合**は「強め」「弱め」など感覚値で書いてもOK。スキルが目安値に変換する。
- **雑魚敵IDが不明な場合**は省略してOK。スキルが `作品別雑魚敵使用状況調査.md` を参照して主力敵を採用する。
