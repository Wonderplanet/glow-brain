# インゲーム要件テキスト テンプレート

スキル（`/masterdata-ingame-creator`）に渡す要件テキストのテンプレート。
✅ = 必須 / 〇 = 任意（省略するとスキルが自動補完）

---

## テンプレート A：dungeon（限界チャレンジ）用

### 通常ブロック（dungeon_normal）

| 項目 | 値 | 必須 | 備考・選択肢 |
|------|----|:----:|------------|
| コンテンツ種別 | `dungeon` | ✅ | 固定値 |
| シリーズID | `{spy}` | ✅ | spy / kai / dan ... |
| 雑魚敵（主力） | `{enemy_xxx_00001}` | ✅ | `enemy_xxx` / `chara_xxx` |
| 雑魚敵（サブ） | `{enemy_xxx_00101}` | 〇 | 不要なら省略 |
| 雑魚 HP | `{1,000}` | ✅ | 目安: 1,000〜10,000 |
| 雑魚 攻撃力 | `{5,000}` | ✅ | 目安: 200〜3,000 |
| 雑魚 移動速度 | `{少し速い}` | ✅ | 遅い / 普通 / 少し速い / 速い |
| 雑魚 必殺ワザ | `{なし}` | ✅ | あり / なし |
| コマ効果 | `{なし}` | 〇 | None / AttackPowerDown / Poison / Darkness / Burn / Gust |
| 背景アセット | `{spy_00005}` | 〇 | 不明なら省略（既存シリーズから引用） |
| BGM | `{SSE_SBG_003_002}` | 〇 | 不明なら省略 |

> **固定値（記述不要）:** アウトポスト HP = 100 / コマ行数 = 3行

## 参考値

### 移動速度

| 表現 | move_speed（内部値） |
|------|-------------------|
| 遅い | 20〜35 |
| 普通 | 35〜50 |
| 少し速い | 45〜55 |
| 速い | 50〜80 |

### HP・攻撃力の目安

| 種別 | HP目安 | 攻撃力目安 |
|------|--------|---------|
| 雑魚（通常難易度） | 1,000〜10,000 | 200〜500 |
| 雑魚（高難易度） | 50,000〜300,000 | 500〜2,000 |
| ボス（標準） | 100,000〜500,000 | 300〜3,800 |
| dungeon 雑魚 | 1,000〜5,000 | 5,000前後（高設定） |
| dungeon ボス | 50,000〜100,000 | 10,000〜20,000 |

## 省略できるもの（スキルが自動補完）

| 省略した場合の扱い |
|-----------------|
| **キャラID不明** → 名前（例:「ロイド」）で記述すれば、スキルがIDを調査する |
| **アセット不明** → 同シリーズの既存ステージから自動引用する |
| **数値が曖昧** → 「強め」「弱め」など感覚値で書けば、スキルが目安値に変換する |
| **雑魚敵ID不明** → `作品別雑魚敵使用状況調査.md` を参照して主力敵を自動採用する |
