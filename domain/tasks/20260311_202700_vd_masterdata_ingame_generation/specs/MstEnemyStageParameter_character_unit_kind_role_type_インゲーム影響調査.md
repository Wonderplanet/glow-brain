# MstEnemyStageParameter: character_unit_kind / role_type インゲーム影響調査

## 概要

`character_unit_kind` と `role_type` は「表示用のみ」ではなく、**実際のインゲームプレイに影響するパラメータ**。

---

## role_type → ダメージ計算に直接影響（影響度: 高）

### 仕組み

```
MstEnemyStageParameter.role_type
  └─ MstUnitRoleBonus テーブルをルックアップ
       ├─ color_advantage_attack_bonus  （色有利時の攻撃倍率）
       └─ color_advantage_defense_bonus （色不利時の被ダメ倍率）
```

- `role_type` に応じて `MstUnitRoleBonus` から色相性ボーナス係数を引き、**攻撃力計算・被ダメ計算の両方に乗算**される
- `AttackPowerCalculator.cs` で `colorAdvantageAttackBonus` を攻撃力に掛け算
- `HPCalculator.cs` で `colorAdvantageDefenseBonus` を受けたダメージに掛け算

**`role_type` が変わると、同じ敵でも与ダメ・被ダメが変わる。**

### 実データ例（MstUnitRoleBonus.csv）

| role_type | color_advantage_attack_bonus | color_advantage_defense_bonus |
|-----------|------------------------------|-------------------------------|
| Attack    | 1.5                          | 0.7                           |
| Defense   | 1.5                          | 0.7                           |
| Support   | 1.5                          | 0.7                           |
| Technical | 1.5                          | 0.7                           |

### 有効な enum 値

`None`, `Attack`, `Balance`, `Defense`, `Support`, `Unique`, `Technical`, `Special`

### VDブロック作成時の注意

雑に `None` や全部 `Attack` にすると色相性ボーナスの乗算係数が変わり、難易度設計が崩れる可能性がある。
既存の同作品キャラの設定を `vd_all/data/MstEnemyStageParameter.csv` で参照して合わせること。

---

## character_unit_kind → 演出とレーン配置に影響（影響度: 中）

### 仕組み

```
character_unit_kind（Normal / Boss / AdventBattleBoss）
  ├─ IsBoss フラグ（Boss または AdventBattleBoss のとき true）
  ├─ → MarchingLaneDirector でレーン振り分け（ボスは専用レーン）
  └─ → MstAutoPlayerSequence.aura_type と連動（出現時VFX/SE演出）
```

#### レーン配置

`CharacterUnitFactory.cs` が `IsBoss` フラグを見てマーチングレーンを振り分ける。
ボスと雑魚で**移動ルート・配置が変わる**。

#### オーラ演出

MstAutoPlayerSequence の `aura_type` カラムと対応し、出現エフェクトが変わる。

| aura_type    | 用途                           |
|--------------|--------------------------------|
| Default      | 通常（雑魚敵・通常ボス）       |
| Boss         | イベントボスクラス             |
| AdventBoss1  | 降臨バトル wave1 のボス        |
| AdventBoss2  | 降臨バトル wave2〜3 のボス     |
| AdventBoss3  | 降臨バトル最終 wave（最大演出）|

#### HP/ATK への影響は無し

HP・ATK などのステータス調整はしない。AI行動パターンの明示的な分岐もコード上では見当たらない。

### 有効な enum 値

`Normal`, `Boss`, `AdventBattleBoss`（`Rare` は C# 定義にあるが実運用上は未使用）

---

## テーブル依存関係

```
MstEnemyStageParameter.role_type
  └─ MstUnitRoleBonus → 攻撃倍率・防御倍率（ダメージ計算）

MstEnemyStageParameter.character_unit_kind
  └─ CharacterUnitModel.IsBoss
       ├─ MarchingLane 配置ロジック（CharacterUnitFactory.cs）
       └─ MstAutoPlayerSequence.aura_type（出現演出）
```

---

## まとめ

| カラム | 表示のみ？ | 実際の影響 |
|--------|-----------|-----------|
| `role_type` | ❌ 違う | 色相性ダメージ計算（`MstUnitRoleBonus` 経由）に**直結** |
| `character_unit_kind` | ❌ 違う | レーン配置 + 出現演出（演出ロジックと連動）に影響 |
