# ロイド「毒軽減」特性 設定箇所調査

**調査日**: 2026-03-15
**対象キャラ**: 〈黄昏〉ロイド（`chara_spy_00101`）
**調査目的**: MstKomaLine の `koma_effect_type=Poison` によるデバフを軽減する特性がどこで設定されているかを明らかにする

---

## 結論

毒軽減は **MstUnit → MstUnitAbility → MstAbility** の3テーブル連携で設定されている。

```
MstUnit.mst_unit_ability_id1
  → MstUnitAbility.id
    → MstAbility.ability_type = "PoisonDamageCut"
```

## クライアント実装

### 定数定義

- `UnitAbilityType.cs`: `PoisonDamageCut` として定義
- `StateEffectType.cs`: `PoisonDamageCut = 24` として定義

### 発動ロジック（UnitAbilityModel.cs）

- `PoisonDamageCut` は「召喚直後に状態変化を付与される特性」リストに含まれる → **召喚後に常時発動**
- 発動時に `StateEffectType.PoisonDamageCut` の状態効果を `Parameter1`（=20）の値で生成

```csharp
UnitAbilityType.PoisonDamageCut
  => CreateStateEffect(StateEffectType.PoisonDamageCut, Parameter1)
```

---

## 毒軽減の効果メカニズム

1. ユニット召喚時に `StateEffectType.PoisonDamageCut`（値=20）の状態効果が付与される
2. MstKomaLine で `koma_effect_type=Poison` のコマ効果が発動すると毒ダメージが発生
3. クライアントが該当ユニットの `PoisonDamageCut` 状態効果を参照し、ダメージを20%軽減して適用する

---

## 新規キャラに毒軽減を設定する場合

1. **MstAbility** に `ability_type=PoisonDamageCut` のレコードが既に存在するので、新規追加不要
2. **MstUnitAbility** に新しい行を追加（`mst_ability_id=ability_poison_damage_cut`、`ability_parameter1`に軽減率を設定）
3. **MstUnit** の `mst_unit_ability_id{N}` に上記で作成したIDを設定する
