## 目的

メインセッションをオーケストレーター専念とし、サブエージェントに個別ブロックの作業を委譲することで、
**16作品 × 2ブロック（Normal/Boss）= 32ブロック**分のVDインゲームマスタデータを効率的に生成する。

## 実行フロー

```
メインセッション（オーケストレーター）
├── [現状確認] vd-ingame-design-creator/ をスキャン
│
├── [Phase 1] design.md 一括作成（未作成ブロックのみ）
│   └── 3ブロック × サブエージェント並列起動 × 繰り返し
│
├── [Phase 2] CSV + design.json 一括生成（未生成ブロックのみ）
│   └── 3ブロック × サブエージェント並列起動 × 繰り返し
│
└── [Phase 3] 全体統合xlsx生成
```

---

## Phase 1: design.md 一括作成

### 手順

1. **対象ブロックの特定**

   `vd-ingame-design-creator/` 配下のフォルダを確認し、`design.md` が存在しないブロックをリストアップする。

   - `vd_all/` は除外する
   - 上記16作品 × 2種別（Normal/Boss）= **最大32ブロック**が対象
   - すでに `design.md` が存在するブロックはスキップする

2. **サブエージェント並列起動（3ブロックずつ）**

   未作成ブロックを3つずつグループ化し、**1メッセージで3つのTask toolを並列起動**する。
   全サブエージェントの完了を待ち、次のグループへ進む。

   **各サブエージェントへの指示**:

   ```
   以下のスキルをバッチモードで実行してください:
   /vd-masterdata-ingame-design-creator 作品ID={作品ID} ブロック種別={Normal|Boss} --batch

   完了したら "DONE: {ブロックID}" とだけ出力してください。

   作業ディレクトリ:
   /Users/junki.mizutani/Documents/workspace/glow/glow-brain-repos/glow-brain-shuna
   ```

   **Task toolパラメータ**:
   - `subagent_type`: `general-purpose`
   - 3ブロック分を同一メッセージ内で並列起動（依存関係なし）

3. **全ブロックのdesign.mdが完成したらPhase 2へ**

---

## Phase 2: CSV + design.json 一括生成

### 手順

1. **対象ブロックの特定**

   `vd-ingame-design-creator/` 配下のフォルダを確認し、`generated/` フォルダが存在しない or CSVが未生成のブロックをリストアップする。

   - `design.md` が存在しないブロックはスキップ（Phase 1未完了）
   - `generated/MstInGame.csv` が存在すればCSV生成済みとみなす

2. **サブエージェント並列起動（3ブロックずつ）**

   未生成ブロックを3つずつグループ化し、**1メッセージで3つのTask toolを並列起動**する。
   全サブエージェントの完了を待ち、次のグループへ進む。

   **各サブエージェントへの指示**:

   ```
   以下のスキルを順番に実行してください:

   ① /vd-masterdata-ingame-data-creator ブロックID={ブロックID}
   ② /vd-masterdata-ingame-design-json-creator ブロックID={ブロックID}

   完了したら "DONE: {ブロックID}" とだけ出力してください。

   作業ディレクトリ:
   /Users/junki.mizutani/Documents/workspace/glow/glow-brain-repos/glow-brain-shuna
   ```

   **Task toolパラメータ**:
   - `subagent_type`: `general-purpose`
   - 3ブロック分を同一メッセージ内で並列起動（ブロック間に依存関係なし）
   - ①と②は同一サブエージェント内で順次実行

3. **全ブロックのCSV+design.jsonが完成したらPhase 3へ**

---

## Phase 3: 統合xlsx生成

全ブロックの生成物（design.json + generated CSV）を1つのxlsxに統合する。

```
/vd-masterdata-ingame-xlsx-creator
```

## 注意事項

- サブエージェントは `general-purpose` タイプを使用する
- バッチモード（`--batch`）はサブエージェント専用。メインセッションでは通常モードを使う
- 各ラウンドはサブエージェント全員の完了を待ってから次に進む（待機してよい）
- エラーが発生したブロックは個別にリトライする
- `vd_osh_boss_00001` は design.md 済み・CSV未生成なので Phase 2 のみ対象
