# 地獄楽分のガシャプレハブ作成とテンプレート作成

*   業務フローの流れ
今後、監修提出への影響ややり取りの手間が少ないようにしたい。
→ガシャTOPの画像をクリエイティブ班で作成し、監修提出
→UI班は作成されたPSDを基にプレハブ化
　→安定してきたら、プレハブ組み込みもバナー班にお願いしても良さそう
