# ピックアップガシャの扉絵が切り替わる時のトランジションが資料と相違している

【発生症状】
ピックアップガシャのガシャ一覧TOP画面で、扉絵が切り替わる時、
トランジションが資料と実機で相違しています。

＜相違箇所＞
・資料：扉絵の左下から右上に向かうモーショングラフィックで切り替わる
・実機：画面全体がフラッシュして切り替わる

【再現手順】
1.ガシャ一覧TOP画面でピックアップガシャを選択する
2.扉絵をタップする

【補足】
タップによる表示切り替え、時間経過による表示切り替えどちらも
フラッシュによる切り替わり挙動となります。

【再現性】
3回中3回
