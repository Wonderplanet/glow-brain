# いいジャンくじ引換所でカラーメモリー獲得時、アイテムBOX内のアイコンが変わって表示される

【発生症状】
いいジャンくじ引換所で複数のカラーメモリーを獲得した後、
アイテムBOXの強化タブを確認すると
カラーメモリーが本来の属性(色)とは異なるアイコンで表示される場合がございます。

＜一例＞
・「カラーメモリー・ブルー」→「カラーメモリー・レッド」のアイコンで表示される
・「カラーメモリー・イエロー」→「カラーメモリー・ブルー」のアイコンで表示される
・「カラーメモリー・グリーン」→「カラーメモリー・イエロー」のアイコンで表示される

検証時間の都合上、現在確認できている範囲でご報告とさせていただきます。

【再現手順】
1.いいジャンくじ引換所に遷移する
2.くじを引き、カラーメモリーを複数獲得する
3.ホーム＞アイテムBOX＞強化に遷移する
4.カラーメモリーの表示を確認する

【補足】
・カラーメモリーが3種類以上になると発生しやすい傾向となります。
・「消費」「かけら」タブに切り替えると本来のアイコンに戻る挙動となっておりますが、
　再起動をすると発生時の表示に戻っています。
・管理ツールで同じ数のカラーメモリーを付与した場合は、本件が発生しないことを確認しております。

【再現性】
3回中3回発生
