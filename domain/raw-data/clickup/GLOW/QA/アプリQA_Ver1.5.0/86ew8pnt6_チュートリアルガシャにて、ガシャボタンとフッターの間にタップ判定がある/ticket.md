# チュートリアルガシャにて、ガシャボタンとフッターの間にタップ判定がある

【発生症状】
チュートリアルガシャにおいて、ガシャボタン外の下部をタップすると
フッターの中央付近までガシャボタンのタップ判定がある状態となっています。

【再現手順】
1.新規アカウントでチュートリアルガシャまで進行する。
2.ガシャボタンとフッターの「ホーム」ボタンの間をタップする。
3.ガシャボタンが反応することを確認する。

【補足】
なし

【再現性】
3回中3回
