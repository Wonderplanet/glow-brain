# Ver1.5.0のアプリを使用時、スタミナブーストが発生しない

【発生症状】
Ver1.5.0のアプリを使用時、スタミナブーストが設定されているクエストにてスタミナブーストが発生しない状態となっております。

【再現手順】
1.任意のメインクエストを選択し表示物を確認する。
2.任意のメインクエストをクリアし、再バトル時にスタミナブーストが発生しないことを確認する。

【補足】
・develop環境
01.ダンダダン1話で確認

・dev-qa2
各種メインクエストで確認

【再現性】
3回中3回
