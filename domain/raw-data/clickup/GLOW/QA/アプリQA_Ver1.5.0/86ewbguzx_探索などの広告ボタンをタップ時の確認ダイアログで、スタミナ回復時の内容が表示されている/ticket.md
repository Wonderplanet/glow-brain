# 探索などの広告ボタンをタップ時の確認ダイアログで、スタミナ回復時の内容が表示されている

【発生症状】
下記の箇所で、広告ボタンをタップした時の確認ダイアログに、
スタミナ回復の確認ダイアログに表示される
注意文言および回復するスタミナ量が表示されます。

＜該当箇所＞
・探索
・コイン獲得クエスト
・降臨バトル
・ショップ

【再現手順】
※探索の場合
1.ホーム＞探索に遷移する
2.「クイック検索」をタップする
3.「広告を見て無料」をタップする

【補足】
「コンティニュー」「ガシャ」の広告ボタンでは本件が発生しないことを確認しております。

【再現性】
3回中3回発生
