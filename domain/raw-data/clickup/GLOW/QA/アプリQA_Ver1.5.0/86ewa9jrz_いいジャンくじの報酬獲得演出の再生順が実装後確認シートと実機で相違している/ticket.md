# いいジャンくじの報酬獲得演出の再生順が実装後確認シートと実機で相違している

【発生症状】
いいジャンくじの報酬獲得演出の再生順が、
実装後確認シートと実機で相違しています。

＜確認挙動＞
資料：ユニット獲得、原画獲得、所持数上限時の挙動
　　　1. 所持数上限時は「所持上限に達しているため、受け取ることができませんでした。」とダイアログが出ること
　　　2. 1の後アイコンのアニメーションが再生開始になること
　　　3. ユニットを新規獲得した場合、キャラ獲得演出がアイコンのアニメーションの前に表示されること
　　　4. 原画を新規獲得した場合、原画獲得演出がアイコンのアニメーションの前に表示されること

※下記の記載に合わせた場合
　　　「所持時上限ダイアログ」＞「キャラ獲得演出」＞「原画獲得演出」＞
　　　「アイコンアニメーション」＞「キャラのかけら変換アニメーション」

実機：「所持時上限ダイアログ」＞「アイコンアニメーション」＞「キャラのかけら変換アニメーション」＞
　　　「キャラ獲得演出」＞「原画獲得演出」

【再現手順】
1.デバッグメニューからユーザーの所持アイテム付与＆MAXを適用する
2.いいジャンくじ引換所に遷移する
3.6箱目まで「切り替え」を行う
4.引換回数を10回にしてくじを引く
5.獲得演出を確認する

【再現性】
3回中3回発生
