# いいジャンくじ引換所で、くじ実行後の通信中に特定のボタンがタップ可能になっている

【発生症状】
いいジャンくじ引換所で、くじ実行後の通信中に、
下記のボタンがタップ可能になっています。

＜該当箇所＞
・報酬アイコン・・・・・・・・・3回中3回
・「戻る」ボタン・・・・・・・・3回中3回
・ヘッダー・・・・・・・・・・・3回中3回
・「アイテムで引く」ボタン・・・10回中1回
・「ラインナップ一覧」ボタン・・20回中1回
・「切り替え」ボタン・・・・・・20回中1回

【再現手順】
※「アイテムで引く」ボタンの場合
1.いいジャンくじ引換所に遷移する
2.「アイテムで引く」ボタンをタップする
3.「くじを引く」ボタンをタップする
4.くじ結果ダイアログが表示されるまでに「アイテムで引く」ボタンをタップする

【補足】
「アイテムで引く」「ラインナップ一覧」「切り替え」ボタンは、くじ結果ダイアログが表示される直前の
ボタンがグレーアウトからアクティブに変わった瞬間にタップ可能となり、再現します。

【再現性】
「発生症状」に記載
