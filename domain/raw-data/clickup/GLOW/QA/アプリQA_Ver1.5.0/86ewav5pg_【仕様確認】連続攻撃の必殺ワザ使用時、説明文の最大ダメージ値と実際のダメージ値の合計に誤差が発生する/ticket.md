# 【仕様確認】連続攻撃の必殺ワザ使用時、説明文の最大ダメージ値と実際のダメージ値の合計に誤差が発生することについて

【発生症状】
連続攻撃の必殺ワザを使用した際、必殺ワザ説明文に記載の最大ダメージ値と、
実際に与えるダメージ値の合計に若干の誤差が発生します。
説明文の値よりも増える場合、減る場合はありますがこちらは問題ございませんでしょうか。

　▼「元殺し屋の新人教諭 リタ」グレード1の場合
　　・必殺ワザ説明文　　：敵単体に最大3回の連続攻撃(自身の攻撃ステータスの最大120%ダメージ、
　　　　　　　　　　　　　敵が緑属性の場合は自身の攻撃ステータスの最大234%のダメージ)
　　・マスターの攻撃倍率：24% (1ヒット目)
　　　　　　　　　　　　　24% (2ヒット目)
　　　　　　　　　　　　　72% (3ヒット目)
　　　　　　　　　　　　　⇒合計130%
　　・キャラの攻撃力　　：4,044

　　・緑属性以外に対するダメージ
　　　実際のダメージ：
　　　971(1ヒット目) + 971(2ヒット目) + 2,912(3ヒット目) = 4,854

　　　攻撃力4,044の120%ダメージとした場合
　　　4,044 × 1.2 = 4,852.8 ⇒ 4,853 (小数点以下切り上げ)

　　・緑属性に対するダメージ
　　　実際のダメージ
　　　2,839(1ヒット目) + 2,839(2ヒット目) + 8,516(3ヒット目) = 14,194

　　　攻撃力4,044の234%ダメージとした場合
　　　4,044 × 1.5(キラー倍率) × 2.34 = 14,194.44 ⇒ 14,195 (小数点以下切り上げ)

【再現手順】
1.必殺ワザが連続攻撃のキャラで実際の合計ダメージを確認する。
2.攻撃力を元に、必殺ワザ説明文の最大ダメージ値から計算を行う。

なし
