# いいジャンくじ引換所で、開催期間終了を跨いでくじを引いた時、終了確認ダイアログが表示されない

【発生症状】
いいジャンくじ引換所で、開催期間終了を跨いでくじを引いた時、
終了確認ダイアログが表示されずにくじ演出が実行され、
アイテムが表示されていないくじ結果ダイアログが表示されます。

【再現手順】
1.管理ツールで日時を「2029/10/01 14:59」に設定する
2.いいジャンくじ引換所に遷移する
3.終了時間を跨いでくじを引く

【補足】
くじ結果ダイアログは表示されますが、アイテムは獲得しないことを確認しております。

【再現性】
3回中3回
