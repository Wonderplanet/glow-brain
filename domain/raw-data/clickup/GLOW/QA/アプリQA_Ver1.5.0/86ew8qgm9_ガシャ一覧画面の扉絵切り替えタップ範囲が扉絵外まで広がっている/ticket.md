# ガシャ一覧画面の扉絵切り替えタップ範囲が扉絵外まで広がっている

【発生症状】
ガシャ一覧画面の扉絵切り替えタップ範囲が扉絵外まで広がっています。

＜確認箇所＞
・扉絵の上下
・ガシャを引くボタンの周囲

【再現手順】
1.フッター「ガシャ」をタップする
2.ガシャ一覧画面で扉絵が切り替わるガシャを選択する
3.該当箇所をタップしてページ切り替えが行われることを確認する

【補足】
資料には「扉絵は扉絵をタップすると切り替わる」と記載されています。

【再現性】
3回中3回
