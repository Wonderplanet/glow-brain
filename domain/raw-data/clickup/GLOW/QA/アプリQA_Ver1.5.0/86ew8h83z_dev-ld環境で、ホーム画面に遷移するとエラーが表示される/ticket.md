# dev-ld環境で、ホーム画面に遷移するとエラーが表示される

【発生症状】
dev-ld環境で、ホーム画面へ遷移するとエラーが表示され、
その後、ガシャ画面へ遷移するとローディングから進まずに進行不能となります。

【再現手順】
1.ホームに遷移する
2.エラーダイアログで「エラーを無視」をタップする
3.フッターの「ガシャ」をタップする

【再現性】
3回中3回発生

【環境情報】
接続環境：dev-ld
検証ビルド：iOS #24(566)
　　　　　　aOS #24(567)
検証端末：iPhone XS (iOS:17.5.1)
　　　　　AQUOS R2(706SH) (aOS:10.0)
