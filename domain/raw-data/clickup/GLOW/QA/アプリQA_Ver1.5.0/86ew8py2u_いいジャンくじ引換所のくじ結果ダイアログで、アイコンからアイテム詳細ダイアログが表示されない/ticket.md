# いいジャンくじ引換所のくじ結果ダイアログで、アイコンからアイテム詳細ダイアログが表示されない

【発生症状】
いいジャンくじ引換所のくじ結果ダイアログで、
アイコンを長押しした時、アイテム詳細ダイアログが表示されません。

【再現手順】
1.いいジャンくじ引換所に遷移する
2.くじを引く
3.くじ結果ダイアログのアイテムアイコンを長押しする

【補足】
・Confluence資料には下記の記載を確認しております。
　「アイテム詳細画面ダイアログはゲーム内のアイテムアイコンが表示される箇所で、
　　アイテムアイコンをタップ時に必ず表示します。」
・類似の画面となるプレミアムメダルガシャは長押しでアイテム詳細が表示されることを確認しております。

【再現性】
3回中3回発生
