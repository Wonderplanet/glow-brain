# 「地獄楽 いいジャン祭ピックアップガシャ」にて、山田浅ェ門 桐馬のキャラ訴求文に誤りがある

【発生症状】
「地獄楽 いいジャン祭ピックアップガシャ A」、および「地獄楽 いいジャン祭ピックアップガシャ B」において、
ガシャTOPで表示される山田浅ェ門 桐馬のキャラ訴求文に『必殺ワザの攻撃で火力に貢献する』とありますが、
実際の必殺ワザはバフのみで攻撃を行う効果はないため、誤った内容となっています。

　▼キャラ訴求文
　=================
　味方を守りつつ、
　必殺ワザの攻撃で
　チームの火力に
　貢献するぞ!
　=================

　▼必殺ワザ文言 ※グレード1
　==============================
　自身と同じコマと一つ先までのコマにいる緑属性・無属性の味方と自身の攻撃を6秒間、10%UP
　 + 自身と同じコマと一つ先までのコマにいる緑属性・無属性の味方と自身の被ダメージを6秒間、10%カット
　==============================

【再現手順】
1.「地獄楽 いいジャン祭ピックアップガシャ」のガシャTOPに遷移する。
2.山田浅ェ門 桐馬のキャラ訴求文を確認する。
3.山田浅ェ門 桐馬の性能詳細>必殺ワザタブに遷移する。
4.必殺ワザの文言を確認する。
