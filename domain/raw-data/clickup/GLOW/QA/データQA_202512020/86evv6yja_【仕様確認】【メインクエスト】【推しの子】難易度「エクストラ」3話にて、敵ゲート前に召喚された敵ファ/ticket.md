# 【仕様確認】【メインクエスト】【推しの子】難易度「エクストラ」3話にて、敵ゲート前に召喚された敵ファントムに近づいても行動を起こさない事について

【発生症状】
【メインクエスト】【推しの子】難易度「エクストラ」3話にて、
敵ゲート前に召喚された敵ファントムに近づいても行動を起こしませんが、
こちらは意図した動作となりますでしょうか。
味方キャラが一方的に攻撃できてしまうため、念のための確認とさせて頂きます。

追記：
難易度「ハード」2話の敵ゲート前や3行目に召喚された一部敵ファントム、
難易度「ノーマル」2話の敵ゲート前に召喚された敵ファントムに近づいた場合も、
行動を起こさないことを確認いたしました。

【補足】
味方キャラを召喚（召喚された敵ファントムから離れた位置のコマ）することで、
該当敵ファントムが行動を開始することを確認しております。

【再現手順】
1.管理ツールで日時を「2026/1/1 00:00:00 ～」に設定する
2.【メインクエスト】【推しの子】難易度「エクストラ」3話を選択する
3.コマ3行目まで進行し、敵ゲート前に召喚された敵ファントムが行動を起こさないことを確認する
4.味方キャラを召喚すると（3）の敵ファントムが行動開始することを確認する
