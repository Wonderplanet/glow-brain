# 降臨バトルで再挑戦ボタンをタップするとエラーが表示される

【発生症状】
降臨バトルのリザルト画面で、
再挑戦ボタンをタップするとエラーが表示されます。

【再現手順】
1.任意の降臨バトルに挑戦する
2.リザルト画面まで進行する
3.再挑戦ボタンをタップする
