# 【仕様確認】プレミアムパスの無料挑戦枠では再挑戦ボタンが有効にならない点について

【確認内容】
プレミアムパス適用中の無料挑戦枠について
現在、再挑戦ボタンの対象にならないことを確認しております。

資料には「広告視聴挑戦枠」では再挑戦ボタンが表示されないと記載がありましたが
プレミアムパス適用中の無料挑戦枠も同じ扱いとなりますでしょうか。

可能であれば、通常の無料挑戦枠と同様に再挑戦できる方がユーザーライクかと思いますが
工数などの都合もあるかと思いますので、一度ご確認をお願いいたします。

＜確認モード＞
・コイン獲得クエスト
・降臨バトル

【確認手順】
1.コイン獲得クエストか降臨バトルのプレミアムパス無料挑戦枠が2以上残っている状態で挑戦する
2.バトルリザルトまで進行する
3.再挑戦ボタンが表示されないことを確認する

【補足】
通常の無料挑戦枠＞再挑戦ボタン表示と
広告視聴挑戦枠＞再挑戦ボタン非表示は資料通りの挙動であることを確認しています。
