# 交換所で開催期間の終了を跨ぎ商品を交換した時、エラーが表示される

【発生症状】
交換所で、開催期間の終了を跨いで商品を交換した時、エラーが表示されます。

【再現手順】
1.管理ツールで日時をイベント交換所の終了日時に設定する
2.交換所一覧＞イベント交換所をタップする
3.開催期間終了を跨ぎ、任意のアイテムを交換する

【補足】
開催期間の終了を跨いで交換所バナーをタップした時は
開催期間が終了した旨のダイアログが表示されて、ホームに遷移することを確認しております。

【再現性】
3回中3回発生
