# 「推しの子交換所」のラインナップが本開催時と復刻時で連動している

【発生症状】
「推しの子交換所」で任意のアイテム交換後、
復刻「推しの子交換所」のラインナップも連動して交換可能数が減っております。

【再現手順】
※本開催時と復刻が同時開催されている時間「2025-11-02 12:00:00」に設定しておく
1.交換所一覧画面＞「推しの子交換所」に遷移する
2.任意のアイテムを交換する
3.復刻「推しの子交換所」に遷移し確認する

【補足】
「推しの子交換所」の期間終了後、
復刻の「推しの子交換所」ではアイテムの交換期限が「残り1分未満」と表示され続けており、
アイテムの交換を試みると、交換期限が終了した旨のダイアログが表示されることを確認しております。

【再現性】
3回中3回発生
