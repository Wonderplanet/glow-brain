# 交換所の交換可能数が「無制限」のアイテムで、交換確認ダイアログの挙動が「制限あり」の場合と相違している

【発生症状】
交換所の交換可能数が「無制限」のアイテムで、
交換に必要なアイテムが「0」の時、
交換確認ダイアログの挙動が「制限あり」の場合と相違しています。

＜確認内容＞
▼制限あり
・交換数は「1」が表示される
・「+1」「+10」「最大」ボタンがアクティブ
・ボタンをタップすると交換数がボタン通りに増減する
・交換数が「1」以上の時「交換する」をタップすると「アイテムが不足しています。」と表示される

▼無制限
・交換数は「1」が表示される
・「最小」「-10」「-1」「+1」「+10」「最大」ボタンがグレーアウトしている
・「+1」「+10」「最大」ボタンをタップすると交換数が「0」になる
・「最小」「-10」「-1」ボタンをタップすると交換数が「1」になる
・交換数が「1」の時に「交換する」をタップすると「アイテムが不足しています。」と表示される
・交換数が「0」の時に「交換する」をタップすると「交換上限に達しています。」と表示される

【再現手順】
1.管理ツールで交換に必要なアイテムを「0」にする
2.ホーム＞交換所＞上から4番目の交換所(交換所02)をタップする
3.無制限の「コインｘ1」をタップする
