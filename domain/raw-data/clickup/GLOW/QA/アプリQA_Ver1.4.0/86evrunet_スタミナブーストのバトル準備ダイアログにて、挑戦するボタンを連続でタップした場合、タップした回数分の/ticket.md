# スタミナブーストのバトル準備ダイアログにて、挑戦するボタンを連続でタップした場合、タップした回数分のスタミナが消費される

【発生症状】
スタミナブーストのバトル準備ダイアログにて、挑戦するボタンを連続でタップした場合、
勝利後やギブアップ後にタップをした回数分のスタミナが消費されている状態となっております。

【再現手順】
1.任意のメインクエストのステージをクリア済みにする。
2.クリア済みのステージでバトルボタンをタップし、スタミナブーストのバトル準備ダイアログを表示する。
3.回数を1回に設定し、挑戦するボタンを連続で素早くタップする。
4.勝利、敗北、ギブアップのいずれかでバトルを終了する。
5.スタミナが余分に消費されていることを確認する。

【補足】
消費するスタミナが不足する状態になった場合、バトル遷移中に「スタミナが不足しています。」と表示される。

【再現性】
3回中3回
