# 交換所の交換確認ダイアログで、アイテム名が表示されていない

【発生症状】
交換所の交換確認ダイアログで、アイテムアイコンの下に
アイテム名が表示されておりません。
こちら資料上の画像では、該当箇所にアイテム名が表示されているため相違しています。

＜確認箇所＞
・通常交換所
・イベント交換所

【再現手順】
1.ホームで「交換所」をタップする
2.交換所一覧画面で任意の通常交換所かイベント交換所のバナーをタップする
3.交換所ラインナップ画面で任意のアイテムをタップする

【補足】
既存機能を移動させた「キャラのかけらBOX交換所」ではアイテム名が表示されていました。

【再現性】
3回中3回
