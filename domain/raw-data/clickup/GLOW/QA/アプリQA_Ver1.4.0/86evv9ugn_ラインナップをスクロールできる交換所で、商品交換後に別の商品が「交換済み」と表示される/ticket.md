# ラインナップをスクロールできる交換所で、商品交換後に別の商品が「交換済み」と表示される

【発生症状】
ラインナップをスクロールできる交換所で、商品交換後に画面スクロールを行うと
別の商品が「交換済み」と表示される不具合が発生しています。

※コイン交換所では再現しなかったため、下記の過去不具合と同様に
　スクロール挙動が原因の可能性があると考えております。
　└https://app.clickup.com/t/86erw6dg5

【再現手順(一例)】
1.【推しの子】いいジャンメダル交換所に遷移する
2.画面下へスクロールし「有馬かな推し！(エンブレム)」を交換する
3.画面上へスクロールし未交換の「プリズム50個」が「交換済み」となっていることを確認する
※他の商品購入や、下にスクロールした場合でも現象が発生します。

【補足】
・不具合で「交換済み」となったアイテムの入手はしていません。
・交換所に入り直すと表示が元に戻ります。
