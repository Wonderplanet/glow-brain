# キャラ詳細の必殺ワザタブ内にて、一部キャラの必殺ワザ文言で文字間の表示不具合が発生している

【発生症状】
キャラ詳細の必殺ワザタブにおいて、一部キャラの必殺ワザ文言で以下の表示不具合が発生しています。

　①文字間が広くなっている
　　該当キャラ：阿波連 れいな/甘戸 めめ/和倉 優希

　②「+」の前後の文字間が狭く、視認がし辛くなっている
　　該当キャラ：B小町不動のセンター アイ/勇気を纏うコスプレ 乃愛　など ※文字数が多い必殺ワザ

　③「+」の前後の文字間が広くなっている
　　該当キャラ：愛を届ける聖夜のサンタ 橘 美花莉/綱代 慎平

【再現手順】
1.上記該当キャラのキャラ詳細に遷移する。
2.必殺ワザタブをタップし、必殺ワザ文言の表示を確認する。

【補足】
※グレードアップ確認/グレードアップ成功ダイアログでは問題ありませんが、
　必殺ワザ詳細ダイアログでも同様の現象が発生しています。
