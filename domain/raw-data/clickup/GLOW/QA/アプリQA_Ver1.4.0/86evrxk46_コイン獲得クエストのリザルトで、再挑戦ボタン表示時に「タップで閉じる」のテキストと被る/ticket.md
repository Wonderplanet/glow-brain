# コイン獲得クエストのリザルトで、再挑戦ボタン表示時に「タップで閉じる」のテキストと被る

【発生症状】
コイン獲得クエストのバトルリザルトで
再挑戦ボタンが表示されていると、UIが「タップで閉じる」のテキストと被っています。

【再現手順】
1.コイン獲得クエストの無料挑戦枠が2以上残っている状態で挑戦する
2.バトルリザルトまで進行する

【補足】
・従来Ver.ではコイン獲得クエストに「タップで閉じる」の表示がありませんでしたが
　v1.4.0では表示されるようになっていたため、発生している状況です。
・iOS、Androidで発生することを確認しています。
・下記モードや画面で再挑戦ボタンが表示された場合は問題ありませんでした。
　└メイン・イベントクエスト、降臨バトル、コンティニュー
