# アイテムBOXの「かけら」タブで、かけらの表示順が資料と相違している

【発生症状】
アイテムBOXの「かけら」タブで、かけらの表示順が資料と実機で相違しております。

＜相違箇所＞
▼資料(該当箇所を抜粋)
また、表示順をキャラ一覧のデフォルトの表示順に合わせて表示する内容に変更します。（できたら）

▼実機
キャラ一覧のデフォルトの表示順(レアリティ順)になっていない

【再現手順】
