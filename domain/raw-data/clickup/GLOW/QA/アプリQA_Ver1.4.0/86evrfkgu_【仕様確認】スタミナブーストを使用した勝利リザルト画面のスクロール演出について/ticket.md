# 【仕様確認】スタミナブーストを使用した勝利リザルト画面のスクロール演出について

【発生症状】
スタミナブーストを使用して勝利リザルト画面が表示された時、
報酬エリアに周回数分の報酬演出が自動スクロールしながら表示されますが、
報酬の量によっては、自動スクロールが報酬表示演出より先行して進み、
白い背景の報酬エリアが表示されてしまう状況となります。

こちらに関しまして、現在の表示挙動で問題ありませんでしょうか。
